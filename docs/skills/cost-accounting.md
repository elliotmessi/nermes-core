# 成本核算技能

## 技能描述

成本核算技能用于自动计算企业产品或服务的生产成本，支持品种法、分批法、分步法等多种成本核算方法。系统根据料、工、费的归集与分配规则，自动完成直接材料、直接人工和制造费用的归集，并按设定的分配标准（如产量、工时、机器工时等）将制造费用分摊到各成本对象。最终输出产品成本计算单和成本分析报告。

## 适用场景

- **制造业成本核算**：按月核算各类产品的材料成本、人工成本和制造费用，计算单位产品成本
- **项目成本追踪**：按项目维度归集成本，实时监控项目预算执行情况
- **服务业成本分析**：按服务项目或客户维度核算交付成本，为定价提供依据
- **标准成本与实际成本对比**：设置标准成本，期末分析实际成本与标准成本的差异及原因

## 输入格式

```json
{
  "costing_method": "品种法",
  "period": "2026-04",
  "cost_objects": [
    {"product_code": "A001", "product_name": "智能网关", "quantity": 500},
    {"product_code": "A002", "product_name": "传感器模组", "quantity": 2000}
  ],
  "direct_materials": [
    {"material_code": "M001", "amount": 150000.00, "used_by": {"A001": 100000, "A002": 50000}}
  ],
  "direct_labor": {"total": 80000.00, "allocation_basis": "工时", "total_hours": 4000},
  "manufacturing_overhead": {"total": 60000.00, "allocation_basis": "机器工时"}
}
```

## 输出示例

```
产品成本计算单
期间：2026年4月

产品：智能网关（A001）    产量：500台

成本项目        总成本        单位成本
──────────────────────────────────────
直接材料       100,000.00     200.00
直接人工        40,000.00      80.00
制造费用        30,000.00      60.00
──────────────────────────────────────
生产成本合计   170,000.00     340.00

制造费用分配明细：
  分配标准（机器工时）：2,000小时
  分配率：60,000 / 4,000 = 15.00元/小时
  A001 分配：15.00 × 2,000 = 30,000.00元

实际成本 vs 标准成本差异：
  材料价格差异：-2,000.00（有利）
  人工效率差异： +500.00（不利）
  费用预算差异：-1,200.00（有利）
```

## 注意事项

- 成本核算前需确保所有生产领料单和工时记录已完整录入系统
- 选择分配标准时需考虑成本动因的相关性，不合理的分配标准会扭曲成本信息
- 在产品与完工产品成本的划分需根据企业生产工艺特点选择合理方法
- 成本核算周期应与会计期间保持一致，避免跨期混入
- 建议定期进行成本差异分析，及时发现异常波动并采取纠正措施
