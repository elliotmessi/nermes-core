# 常见业务会计分录速查表

> 涵盖企业日常经营中最常见的会计分录，适合新手快速查阅。所有分录以**一般纳税人**为例，小规模纳税人的差异在文末单独说明。

---

## 一、筹资与投资

### 1.1 收到注册资本投入

| 业务场景 | 借 | 贷 |
|---------|----|-----|
| 股东以货币出资 | 银行存款 | 实收资本——XX股东 |
| 股东以固定资产出资（按公允价值） | 固定资产<br>应交税费——应交增值税（进项税额） | 实收资本——XX股东 |
| 资本溢价（多投部分） | 银行存款 | 实收资本<br>资本公积——资本溢价 |

> 示例：A公司注册资本100万元，新股东投入150万元，占股30%。
> 借：银行存款 150万
> 贷：实收资本——新股东 30万（100万×30%）
> 资本公积——资本溢价 120万

### 1.2 银行借款

| 业务场景 | 借 | 贷 |
|---------|----|-----|
| 取得短期借款（≤1年） | 银行存款 | 短期借款 |
| 取得长期借款（>1年） | 银行存款 | 长期借款——本金 |
| 计提及支付短期借款利息 | 财务费用<br>贷：应付利息<br>（支付时）借：应付利息 | 贷：银行存款 |
| 归还借款本金 | 短期借款/长期借款 | 银行存款 |

## 二、采购业务

### 2.1 采购原材料/商品

**一般纳税人（取得增值税专用发票）**：
```
借：原材料/库存商品          （不含税金额）
    应交税费——应交增值税（进项税额）
贷：应付账款/银行存款         （价税合计）
```

**未取得专票或小规模纳税人**：
```
借：原材料/库存商品          （价税合计，不可抵扣）
贷：应付账款/银行存款
```

### 2.2 采购时发生的运费

```
借：原材料/库存商品/销售费用
    应交税费——应交增值税（进项税额）  （运输发票可抵9%）
贷：银行存款
```

### 2.3 预付采购款

**预付时**：
```
借：预付账款
贷：银行存款
```

**收到货物及发票时**：
```
借：原材料/库存商品
    应交税费——应交增值税（进项税额）
贷：预付账款
    （若不足）：银行存款
    （若有余额冲回）：贷：预付账款（红字）或借：银行存款
```

## 三、销售业务

### 3.1 常规销售（一般纳税人，开具专票）

```
借：应收账款/银行存款
贷：主营业务收入            （不含税销售额）
    应交税费——应交增值税（销项税额）
```

> **关键提示**：确认收入必须同步计提增值税销项税，不能"以后开票再交税"。

### 3.2 销售的同时结转成本

```
借：主营业务成本
贷：库存商品
```

> 采用**月末一次加权平均法**计算发出存货的单位成本。

### 3.3 预收货款销售

**预收时**：
```
借：银行存款
贷：合同负债/预收账款
```

**交付商品确认收入时**：
```
借：合同负债/预收账款
贷：主营业务收入
    应交税费——应交增值税（销项税额）
```

### 3.4 商业折扣（直接打折）

按折扣后的净额确认收入，不单独做分录。
> 示例：商品原价100元，打9折收款90元。确认收入=90/(1+税率)，增值税=收入×税率。

### 3.5 现金折扣（早付款给优惠）

采用**净额法**（新CAS 14要求，折扣冲减收入）：
```
确认收入时按全额：
借：应收账款
贷：主营业务收入
    应交税费——应交增值税（销项税额）

客户早付款享受折扣（如2/10）：
借：银行存款          （实收金额）
    主营业务收入       （折扣金额，红字或借方）
贷：应收账款          （原全额）
```

### 3.6 销售退回

```
借：主营业务收入         （红字/负数）
    应交税费——应交增值税（销项税额）（红字）
贷：应收账款/银行存款

同时冲回成本：
借：库存商品
贷：主营业务成本         （红字/负数）
```

## 四、费用报销

### 4.1 差旅费

```
借：管理费用/销售费用——差旅费
    应交税费——应交增值税（进项税额）  （住宿专票可抵6%）
贷：银行存款/其他应付款
```

> 餐费/交通费（打车）通常无进项税抵扣（即使取得专票也不得抵扣）。

### 4.2 业务招待费

```
借：管理费用——业务招待费
贷：银行存款
```

> **税务提醒**：业务招待费按发生额的60%扣除，且不得超过当年销售（营业）收入的5‰。超过部分需纳税调整。

### 4.3 办公用品采购

```
借：管理费用——办公费
    应交税费——应交增值税（进项税额）
贷：银行存款
```

### 4.4 员工报销电话费

```
借：管理费用——通讯费
    应交税费——应交增值税（进项税额）
贷：银行存款
```

> 个人抬头的通讯费发票不可抵扣进项税。

## 五、固定资产业务

### 5.1 购入固定资产

**购入不需安装的设备**：
```
借：固定资产
    应交税费——应交增值税（进项税额）
贷：银行存款
```

**购入需安装设备/房产**：
```
① 购入时：
借：在建工程
    应交税费——应交增值税（进项税额）
贷：银行存款

② 安装调试费、人工费：
借：在建工程
贷：银行存款/原材料/应付职工薪酬

③ 达到预定可使用状态：
借：固定资产
贷：在建工程
```

### 5.2 计提折旧

**按直线法（平均年限法）每月计提**：

生产车间设备：
```
借：制造费用
贷：累计折旧
```

管理部门设备：
```
借：管理费用
贷：累计折旧
```

销售部门设备：
```
借：销售费用
贷：累计折旧
```

### 5.3 固定资产处置

**转入清理**：
```
借：固定资产清理
    累计折旧
    固定资产减值准备（如有）
贷：固定资产
```

**取得处置收入**：
```
借：银行存款
贷：固定资产清理
    应交税费——应交增值税（销项税额）   （一般纳税人13%）
```

**支付清理费用**：
```
借：固定资产清理
贷：银行存款
```

**结转清理净损益**：

净收益：
```
借：固定资产清理
贷：资产处置损益/营业外收入
```

净损失：
```
借：资产处置损益/营业外支出
贷：固定资产清理
```

## 六、工资与薪酬（全流程）

### 6.1 每月计提工资（月末）

```
借：管理费用/销售费用/生产成本/制造费用  （按部门归集）
贷：应付职工薪酬——工资
    应付职工薪酬——社保（单位部分）
    应付职工薪酬——住房公积金（单位部分）
    应付职工薪酬——工会经费           （工资额×2%）
    应付职工薪酬——职工教育经费         （工资额×8%）
```

### 6.2 实际发放工资

```
借：应付职工薪酬——工资              （应发总额）
贷：其他应付款——社保（个人部分）
    其他应付款——住房公积金（个人部分）
    应交税费——应交个人所得税
    银行存款                      （实发金额）
```

### 6.3 缴纳社保和公积金

**缴纳单位部分 + 代扣的个人部分**：
```
借：应付职工薪酬——社保（单位部分）
    应付职工薪酬——住房公积金（单位部分）
    其他应付款——社保（个人部分）
    其他应付款——住房公积金（个人部分）
贷：银行存款
```

### 6.4 缴纳个人所得税

```
借：应交税费——应交个人所得税
贷：银行存款
```

> **每月截止**：工资个税在次月15日前申报缴纳。

## 七、税务相关分录

### 7.1 月末增值税结转

适用于一般纳税人**按月**结转：

**（1）转出当期进项税额**：
```
借：应交税费——应交增值税（转出未交增值税）
贷：应交税费——应交增值税（进项税额）
```

**（2）转出当期销项税额**：
```
借：应交税费——应交增值税（销项税额）
贷：应交税费——应交增值税（转出未交增值税）
```

**（3）结转未交增值税**：

应交（销项>进项）：
```
借：应交税费——应交增值税（转出未交增值税）
贷：应交税费——未交增值税
```

留抵（进项>销项）：
```
借：应交税费——未交增值税
贷：应交税费——应交增值税（转出未交增值税）
```

**（4）次月缴税**：
```
借：应交税费——未交增值税
贷：银行存款
```

### 7.2 附加税计提与缴纳

**计提**：
```
借：税金及附加
贷：应交税费——应交城市维护建设税
    应交税费——应交教育费附加
    应交税费——应交地方教育附加
```

**缴纳**：
```
借：应交税费——应交城市维护建设税
    应交税费——应交教育费附加
    应交税费——应交地方教育附加
贷：银行存款
```

### 7.3 企业所得税

**每季度末计提**：
```
借：所得税费用
贷：应交税费——应交企业所得税
```

**预缴**：
```
借：应交税费——应交企业所得税
贷：银行存款
```

**年末汇算清缴（少补多退）**：

补税：
```
借：以前年度损益调整
贷：应交税费——应交企业所得税
```
```
借：应交税费——应交企业所得税
贷：银行存款
```
```
借：利润分配——未分配利润
贷：以前年度损益调整
```

## 八、月末结转分录

### 8.1 制造费用结转（仅工业企业）

```
借：生产成本——制造费用结转
贷：制造费用
```

### 8.2 完工产品成本结转

```
借：库存商品
贷：生产成本——直接材料
    生产成本——直接人工
    生产成本——制造费用结转
```

### 8.3 损益类科目结转至"本年利润"

**结转收入/利得类**（贷方余额的科目）：
```
借：主营业务收入
    其他业务收入
    投资收益（净收益）
    公允价值变动损益（净收益）
    资产处置损益（净收益）
    营业外收入
贷：本年利润
```

**结转费用/损失类**（借方余额的科目）：
```
借：本年利润
贷：主营业务成本
    其他业务成本
    税金及附加
    销售费用
    管理费用
    财务费用
    营业外支出
    所得税费用
```

> 损益结转后，利润表科目的余额归零。"本年利润"的贷方余额=净利润，借方余额=净亏损。

## 九、年末结转分录

### 9.1 提取盈余公积

```
借：利润分配——提取法定盈余公积     （净利润×10%）
    利润分配——提取任意盈余公积     （股东大会决定）
贷：盈余公积——法定盈余公积
    盈余公积——任意盈余公积
```

> 法定盈余公积累计达到注册资本的50%时，可不再提取。

### 9.2 宣告分配股利

```
借：利润分配——应付现金股利/转作股本的股利
贷：应付股利
```

### 9.3 年末"本年利润"结转至"利润分配"

**盈利时**：
```
借：本年利润
贷：利润分配——未分配利润
```

**亏损时**：
```
借：利润分配——未分配利润
贷：本年利润
```

### 9.4 利润分配内部明细结转

```
借：利润分配——未分配利润
贷：利润分配——提取法定盈余公积
    利润分配——提取任意盈余公积
    利润分配——应付现金股利
```

> 结转后，"利润分配"下除"未分配利润"外的其他明细科目余额归零。

## 十、特殊业务分录

### 10.1 研发支出

**费用化支出**（研究阶段/部分开发阶段）：
```
借：研发支出——费用化支出
贷：银行存款/原材料/应付职工薪酬
```
月末结转：
```
借：管理费用——研发费用
贷：研发支出——费用化支出
```

**资本化支出**（满足条件的开发阶段）：
```
借：研发支出——资本化支出
贷：银行存款/原材料/应付职工薪酬
```
形成无形资产时：
```
借：无形资产
贷：研发支出——资本化支出
```

### 10.2 坏账准备

**计提**（按应收账款一定比例）：
```
借：信用减值损失
贷：坏账准备
```

**实际发生坏账**：
```
借：坏账准备
贷：应收账款
```

**已核销坏账收回**：
```
借：应收账款
贷：坏账准备
```
同时：
```
借：银行存款
贷：应收账款
```

### 10.3 资产减值

```
借：资产减值损失
贷：存货跌价准备/固定资产减值准备/无形资产减值准备
```

> **注意**：存货跌价准备可以转回；固定资产/无形资产/长期股权投资的减值准备**一旦计提不得转回**。

### 10.4 政府补助（总额法）

**与资产相关的补助**：
```
收到时：借：银行存款  贷：递延收益
按期摊销：借：递延收益  贷：其他收益
```

**与收益相关的补助**（补偿已发生费用）：
```
借：银行存款
贷：其他收益/营业外收入
```

## 十一、小规模纳税人分录差异

小规模纳税人**不可抵扣进项税**，采购时价税合计全额计入成本：

**采购**：
```
借：原材料/库存商品          （含税总价）
贷：银行存款
```

**销售**（简易计税，当前税率1%）：
```
借：应收账款
贷：主营业务收入            （含税销售额÷1.01）
    应交税费——应交增值税     （含税销售额÷1.01×1%）
```

**月末增值税结转**：
```
借：应交税费——应交增值税
贷：银行存款
```

## 附：常见科目借贷方向速记

| 科目类型 | 借方（增加） | 贷方（增加） |
|---------|------------|------------|
| 资产类 | ✅ 增加 | 减少 |
| 成本类 | ✅ 增加 | 减少/结转 |
| 费用类 | ✅ 发生 | 结转（转出） |
| 负债类 | 减少 | ✅ 增加 |
| 权益类 | 减少 | ✅ 增加 |
| 收入类 | 结转（转出） | ✅ 实现 |

> **记账口诀**：有借必有贷，借贷必相等。资产=负债+所有者权益。
