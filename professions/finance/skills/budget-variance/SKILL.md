# 预算差异分析

使用本技能时，你需要对企业的**实际经营结果与预算目标进行对比分析，识别差异原因（量差/价差/效率差异），并可编制弹性预算进行动态评估**。

---

## 一、数据准备

### 1.1 所需数据
- **预算数据**：年度/月度预算表（按科目、部门、项目维度）
- **实际数据**：实际发生额（同期口径、同科目、同维度）
- **业务量数据**：实际产量/销量、预算产量/销量、标准工时等
- **预算编制假设**：预算时使用的单价、成本率、费用率等假设参数

### 1.2 数据对齐
- 确保预算科目与实际科目**一一对应**（映射关系明确）
- 预算期间与实际期间**一致**（月度/季度/年度）
- 遇组织架构调整/业务范围变化，需**调整预算基数**再对比

---

## 二、差异基础分析

### 2.1 差异分类

| 差异类型 | 说明 |
|---------|------|
| **有利差异 (Favorable)** | 实际收入 > 预算收入，或实际成本 < 预算成本 |
| **不利差异 (Unfavorable)** | 实际收入 < 预算收入，或实际成本 > 预算成本 |

### 2.2 差异计算
- **绝对差异** = 实际金额 - 预算金额（正=超预算/多收入）
- **相对差异率** = 绝对差异 / 预算金额 × 100%

### 2.3 差异分层
```
公司层面汇总差异
  ├── 各业务部门差异
  │     ├── 收入差异
  │     ├── 成本差异
  │     └── 费用差异
  ├── 各产品线差异
  └── 各费用科目差异
```

## 三、差异原因分析

### 3.1 收入差异分解

**收入差异 = 价格差异 + 销量差异**

- **价格差异** = (实际单价 - 预算单价) × 实际销量
  - 原因：市场竞争调价、产品升级提价、折扣促销
- **销量差异** = (实际销量 - 预算销量) × 预算单价
  - 原因：市场需求变化、竞品冲击、渠道变化、营销效果

### 3.2 成本差异分解

**成本差异 = 用量差异 + 价格差异**

**直接材料差异**：
- **材料价格差异** = (实际单价 - 标准单价) × 实际用量
- **材料用量差异** = (实际用量 - 标准用量) × 标准单价

**直接人工差异**：
- **工资率差异** = (实际工资率 - 标准工资率) × 实际工时
- **人工效率差异** = (实际工时 - 标准工时) × 标准工资率

**制造费用差异**：
- **变动制造费用**：花费差异 + 效率差异
- **固定制造费用**：预算差异 + 产能差异（量差）

### 3.3 费用差异分析

期间费用差异分析维度：
| 费用类别 | 差异原因排查方向 |
|---------|----------------|
| 销售费用 | 广告投放超预算、差旅费控制、佣金政策变化 |
| 管理费用 | 人员编制超预算、办公费用、咨询费 |
| 研发费用 | 研发项目增加/减少、资本化vs费用化选择 |
| 财务费用 | 利率变动、汇兑损益、融资规模变化 |

## 四、弹性预算编制

### 4.1 弹性预算概念
基于**实际业务量**而非预算业务量来重新编制预算，排除"量差"影响，聚焦"价差"和"效率差异"。

### 4.2 弹性预算公式
```
弹性预算收入 = 实际销量 × 预算单价
弹性预算成本 = 实际产量 × 预算单位成本
弹性预算费用(变动) = 实际业务量 × 预算费用率
弹性预算费用(固定) = 原预算固定费用（不变）
```

### 4.3 弹性预算差异分析表

```
| 项目 | 实际结果 | 弹性预算 | 静态预算 | 弹性差异(2)-(3) | 量差(3)-(4) |
|      |    (1)   |    (2)   |    (3)   |      (4)        |    (5)      |
|------|----------|----------|----------|-----------------|-------------|
| 销量 |  实际量  |  实际量  |  预算量  |        -        |       -     |
| 收入 |  实际价×量 | 预算价×实际量 | 预算价×预算量 | 价差         | 量差        |
| 成本 |  实际   |  预算单位成本×实际量 | 预算总成本 | 耗费差异     | 量差        |
| 毛利 |  实际毛利 | 弹性预算毛利 | 静态预算毛利 | 价差+效率差  | 量差        |
```

### 4.4 弹性预算的应用场景
- 实际产量与预算产量偏差大时 → 必须用弹性预算代替静态预算
- 考核生产部门成本控制时 → 用量差剔除产量影响
- 销售部门考核 → 重点是收入价差

## 六、使用规则

- 差异分析必须**先排除量差**（使用弹性预算），再分析价差/效率差
- 差异原因需**归因到具体业务部门**，不可笼统表述
- **重复出现的差异**（连续3个月以上）需升级为制度问题
- **有利差异也要分析原因**（如过度节约可能影响长期发展）
- 报告需区分"可控差异"（内部管理）与"不可控差异"（市场/政策）
- 最终输出应包含**行动建议**，不只是一堆数据
