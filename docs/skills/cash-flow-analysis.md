# 现金流分析技能

## 技能描述

现金流分析技能用于企业现金流量表的自动编制与现金流的深度分析。系统支持直接法和间接法两种编制方式，通过分析企业交易对现金的影响，生成经营活动、投资活动和筹资活动三大现金流分类。同时提供现金流动性分析、自由现金流计算、现金流预测和现金周期分析等高级功能，帮助企业优化资金管理，防范流动性风险。

## 适用场景

- **现金流跟踪与监控**：每周/每月生成现金流量分析报告，掌握企业现金流入流出动态
- **流动性风险预警**：监测经营活动现金流净额的变化趋势，提前发现现金流断裂风险
- **投资决策支持**：计算自由现金流和现金转换周期，评估企业的资金效率和投资回报
- **融资需求评估**：结合未来现金流预测，合理规划融资时点和规模

## 输入格式

```json
{
  "analysis_type": "monthly",
  "period": "2026-04",
  "include_forecast": true,
  "forecast_months": 3,
  "cash_balance": {
    "beginning_balance": 850000.00
  },
  "cash_flow_items": [
    {"category": "operating", "type": "inflow", "item": "销售商品收款", "amount": 1200000.00},
    {"category": "operating", "type": "outflow", "item": "采购付款", "amount": 650000.00}
  ]
}
```

## 注意事项

- 现金流量表的准确编制高度依赖凭证中对现金流量项目的正确分类，需确保操作人员理解分类标准
- 经营活动现金流为负不一定意味着经营不善，高速增长期企业可能出现"盈利但缺钱"的情况
- 自由现金流比净利润更能反映企业的真实盈利质量，建议两者结合分析
- 现金流预测需充分考虑季节性波动和重大资本支出计划
- 建议设置现金余额最低警戒线（如能够覆盖未来3个月的固定支出），低于警戒线自动预警
