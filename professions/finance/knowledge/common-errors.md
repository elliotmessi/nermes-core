# 企业常见会计差错与纠正

> 财务人员最容易犯的错误清单，附正确做法和更正分录。

---

## 一、收入确认类差错

### 1.1 提前确认收入（"寅吃卯粮"）

**错误**：货还没发、客户还没签收就确认了收入。

**正确做法**：确认收入必须满足——客户取得商品控制权（签收/验收）。

**更正分录（跨年发现，重大差错）**：
```
借：以前年度损益调整
    应交税费——应交增值税（销项税额）（冲回已交的销项税）
贷：应收账款
```

### 1.2 滞后确认收入（推迟纳税）

**错误**：货已发、客户已签收，以"还没开票"为由不确认收入。

**正确做法**：收入确认与开票时间无关。不开票也要做无票收入申报。

**更正分录**：
```
借：应收账款
贷：主营业务收入
    应交税费——应交增值税（销项税额）
```

### 1.3 含税收入按不含税记账

**错误**：收款 113 万全部记入"主营业务收入"（忘了拆分增值税）。

**正确做法**：
```
借：银行存款                          113万
贷：主营业务收入                      100万（113÷1.13）
    应交税费——应交增值税（销项税额）   13万
```

## 二、成本费用类差错

### 2.1 资本性支出一次性费用化

**错误**：买了一台 50 万的设备，当期全部计入管理费用。

**正确做法**：应确认为固定资产，分期折旧（5年/10年等）。

**更正分录**：
```
借：固定资产          50万
贷：以前年度损益调整   50万

借：以前年度损益调整      （当期折旧额）
贷：累计折旧              （当期折旧额）
```

### 2.2 费用支出错误资本化

与上面相反：
**错误**：日常维修费 5 万元计入"在建工程"或"长期待摊费用"。

**正确做法**：日常维修应直接费用化。

**更正分录**：
```
借：管理费用/制造费用    5万
贷：在建工程/长期待摊费用   5万
```

### 2.3 业务招待费与差旅费混用

**错误**：请客户吃饭的餐饮发票记入"差旅费"。

**正确做法**：招待客户的餐饮费记入"业务招待费"，且需按60%扣除、上限为收入5‰。

> 错误分类 → 汇算清缴时少调增 → 被税务查到补税+罚款。

### 2.4 员工福利费/工会经费/教育经费不按规定计提

**正确计提方式**：
```
借：管理费用/生产成本等
贷：应付职工薪酬——职工福利费      （工资总额×14%）
    应付职工薪酬——工会经费          （工资总额×2%）
    应付职工薪酬——职工教育经费       （工资总额×8%）
```

**错误**：只在年末实际发生时才计提，甚至不单独设"应付职工薪酬"科目。

## 三、往来款差错

### 3.1 应收账款长期挂账不处理

**情况**：3年前的应收款仍然挂账，对方公司已注销。

**正确做法**：确认坏账，计入损失。

```
借：信用减值损失（以前年度用"资产减值损失"）
贷：坏账准备

确认无法收回后：
借：坏账准备
贷：应收账款
```

> 坏账损失税前扣除需准备：债务人注销证明/破产公告/法院判决/账龄3年以上+催收记录。

### 3.2 "其他应收款"变成"垃圾桶"

**常见乱象**：
- 老板从公司"借款"长期不还 → 视同分红，补缴 20% 个税
- 没有发票的支出全部挂账 → 汇算清缴应费用化，如无发票需调增
- 股东抽逃出资挂账 → 重大法律风险

**清理方法**：每年至少清理一次"其他应收款"，逐笔核实，该收回的收回、该核销的核销、该费用化的费用化。

### 3.3 应付账款忘了含税

**错误**：采购商品时只记了不含税金额。

**正确分录**（一般纳税人，取得专票）：
```
借：原材料/库存商品                     （不含税金额）
    应交税费——应交增值税（进项税额）
贷：应付账款                            （价税合计）
```

## 四、税务差错

### 4.1 进项税额错误抵扣

| 错误 | 正确处理 |
|------|---------|
| 员工餐费发票抵扣进项税 | 集体福利，进项税**不得抵扣**，须转出 |
| 贷款利息专票抵扣进项税 | 贷款服务进项税**不得抵扣** |
| 用于简易计税项目的进项税抵扣 | 须转出或按比例分摊 |
| 个人消费的发票抵扣 | **不得抵扣** |

**进项税额转出分录**：
```
借：相关成本费用/原材料等
贷：应交税费——应交增值税（进项税额转出）
```

### 4.2 印花税漏申报

**常见遗漏**：
- 没签纸质合同 ≠ 不交印花税（电子合同也要交）
- 注册资本没有按"实收资本+资本公积"的合计金额申报
- 借款合同的印花税忘了交
- 框架合同+后续订单的，框架合同和订单都要交

### 4.3 汇算清缴忘记调整

必须纳税调增的项目：
- 行政罚款、税收滞纳金
- 未经核准的资产减值准备
- 超标的业务招待费（发生额60%，上限收入5‰）
- 超标的广告费（一般企业上限收入15%）
- 公益性捐赠超标（上限利润总额12%）
- 与经营无关的支出

## 五、报表差错

### 5.1 现金流量表分类错误

| 业务 | 正确分类 | 常见错误分类 |
|------|---------|------------|
| 购买固定资产 | 投资活动 | 误列为经营活动 |
| 取得银行贷款 | 筹资活动 | 误列为经营活动 |
| 支付股东股利 | 筹资活动 | 误列为经营/投资活动 |
| 银行存款利息收入 | 经营活动（或投资活动） | 误列为筹资活动 |

### 5.2 存货计价方法随意变更

**错误**：今年用先进先出法，明年改加权平均法，后年又改回去。

**正确做法**：存货计价方法一经选定，至少在一个会计年度内保持一致。如需变更须在附注中披露并追溯调整。

## 六、更正差错的正确姿势

### 6.1 当年发现当年差错

直接红字冲销原错误分录，再做正确分录即可。

### 6.2 跨年发现差错

| 差错性质 | 处理方法 |
|---------|---------|
| **不重大** | 计入当期损益，不追溯调整 |
| **重大** | 通过"以前年度损益调整"追溯调整，调整年初留存收益 |

**重大差错的判断**：该差错单独或汇总后可能影响报表使用者的决策。

### 6.3 跨年重大差错更正分录模板

**(1) 调整差错事项**
```
借/贷：相关资产/负债科目
贷/借：以前年度损益调整
```

**(2) 调整所得税影响**
```
借：以前年度损益调整
贷：应交税费——应交企业所得税
（或反向，取决于差错方向）
```

**(3) 结转至留存收益**
```
借：以前年度损益调整（贷方余额）
贷：利润分配——未分配利润
```
