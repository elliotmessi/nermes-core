# 广告投放优化技能 — 直通车/引力魔方/万相台/千川

## 功能描述
提供电商平台广告投放的全流程优化建议，包括计划搭建、选词出价、人群定向、创意优化、效果分析，帮助运营者提升广告 ROI、降低获客成本。

## 核心能力

### 1. 计划搭建
- **投放目标匹配**：测款/拉新/收割/清仓 不同阶段选不同工具
- **预算分配**：新计划日预算30-100元起步，稳定后逐步提升
- **时间折扣**：高转化时段（如20:00-23:00）出价提高至120%-150%

### 2. 关键词策略（直通车/标准计划）
- **选词来源**：生意参谋热搜词、系统推荐词、竞品词、长尾词
- **出价策略**：核心词精准匹配（高价）、长尾词广泛匹配（低价）
- **质量分优化**：影响质量分的因素：点击率（最重要）、相关性、账户历史
- **数据分析**：7天无转化词删除、高ROI词提价、低CTR词优化创意图

### 3. 人群定向（引力魔方/万相台）
- **人群分层**：店铺人群（浏览/收藏/加购/购买） → 竞品人群 → 行业人群 → 跨类目人群
- **拉新目标**：扩展新客人群，出价从低到高测试
- **收割目标**：针对已访未购人群，高溢价投放
- **DMP人群包**：自定义标签组合，精准定向

### 4. 创意优化
- **图片创意**：不同促销文案和配色A/B测试
- **视频创意（抖音千川）**：前3秒黄金法则 + 场景化内容
- **标题文案**：包含核心卖点和促销信息

### 5. 千川投放（抖音电商）
- **计划类型**：短视频引流 / 直播间引流 / 商品卡投放
- **投放方式**：控成本投放 / 放量投放 / 一键起量
- **人群定向**：性别/年龄/地域/行为兴趣标签
- **素材生命周期**：一般3-7天后CTR下降，需持续更新素材

### 6. 效果复盘
- **核心指标**：花费、展现、点击、CTR、CPC、转化数、CVR、ROI、PPC
- **归因分析**：最后一点击 / 多渠道辅助成交价值
- **优化方向**：低ROI计划暂停、高ROI计划加预算、人群包迭代

## 使用场景
- 新品推广计划搭建
- 现有计划优化和调整
- 大促期间广告预算分配策略
- 月度广告投放复盘
- ROI持续下降时的诊断

## 输入
投放平台、商品信息、当前投放数据（可选）、预算范围、投放目标

## 输出
投放策略方案、关键词/人群建议、预算分配方案、创意思路、优化调整建议
