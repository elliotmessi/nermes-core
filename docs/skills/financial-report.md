# 财务报表生成技能

## 技能描述

财务报表生成技能用于根据会计准则和账务数据，自动生成企业标准财务报表及管理报表。支持生成资产负债表、利润表、现金流量表（直接法与间接法）、所有者权益变动表四大主表，以及按管理维度定制化的内部管理报表。系统从科目余额表和凭证数据中自动取数，经过重分类调整和合并抵消后，生成符合会计准则的报表。

## 适用场景

- **月末/年末结账**：结账后自动生成当期财务报表，支持与上期数据进行同比环比分析
- **集团合并报表**：多子公司架构下，自动汇总各公司报表并进行内部交易抵消
- **管理驾驶舱**：按事业部、区域、项目等多维度生成收入、成本、利润分析报表
- **外部审计配合**：生成符合审计要求的报表附注及相关披露信息

## 输入格式

```json
{
  "report_type": "balance_sheet",
  "company_code": "C001",
  "period": "2026-04",
  "include_comparative": true,
  "comparative_period": "2025-04",
  "currency": "CNY",
  "consolidation_scope": ["C001", "C002", "C003"]
}
```

## 注意事项

- 报表生成前务必完成所有期末调整分录（如计提折旧、摊销、坏账准备等）
- 资产负债表必须满足"资产=负债+所有者权益"的恒等式，如不平衡需检查数据源
- 现金流量表的编制对原始凭证的现金流量项目分类准确性依赖较高
- 合并报表的关联交易抵消需提前完成对账，否则会导致重复计算
- 建议在正式报表生成前先运行"报表预检"功能，提前发现数据异常
