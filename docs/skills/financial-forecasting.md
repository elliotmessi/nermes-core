# 财务预测技能

## 技能描述

财务预测技能用于基于历史财务数据和业务驱动因素，运用统计模型和机器学习算法，预测企业未来的财务表现。支持收入预测、成本预测、利润预测、现金流预测和资产负债表预测，预测周期可覆盖月度、季度和年度。系统提供趋势分析、回归分析、时间序列分析等多种预测方法，并结合宏观经济指标和行业数据提高预测精度。

## 适用场景

- **年度经营计划制定**：为下一年度的经营目标和资源配置提供财务数据支撑
- **融资路演准备**：根据业务增长假设生成未来3-5年的财务预测，用于投资机构沟通
- **资金需求规划**：预测未来现金流入流出，规划融资时点、规模和方式
- **并购目标估值**：预测目标公司的未来现金流，为估值模型提供基础数据

## 输入格式

```json
{
  "company_code": "C001",
  "company_name": "XX科技有限公司",
  "base_year": 2025,
  "base_data": {
    "revenue_2025": 8000000.00,
    "cost_2025": 4200000.00,
    "opex_2025": 2500000.00
  },
  "assumptions": {
    "revenue_growth_rate": 0.20,
    "cost_growth_rate": 0.15,
    "inflation_rate": 0.03,
    "tax_rate": 0.25
  },
  "forecast_periods": 3,
  "forecast_granularity": "annual",
  "model_type": "time_series"
}
```

## 注意事项

- 财务预测的准确性高度依赖于假设条件的合理性，建议设置多情景（乐观/基准/悲观）分析
- 历史数据不可完全代表未来，需结合行业趋势、市场竞争格局和宏观经济环境综合判断
- 预测周期越长，不确定性越大，建议远期预测采用更宽泛的置信区间
- 预测模型需要定期回测，将预测值与实际值进行对比分析，持续优化模型参数
- 财务预测应作为决策参考而非绝对依据，重大决策需结合定性分析和专业判断
