# 所得税计提技能

本技能提供企业所得税计提全流程指导，涵盖当期所得税计算、递延所得税确认、暂时性差异识别及账务处理。

---

## 1. 当期所得税计算

### 1.1 基本公式

```
当期所得税 = 应纳税所得额 × 适用税率 － 减免税额 － 抵免税额
应纳税所得额 = 会计利润 ± 纳税调整项目
```

### 1.2 纳税调整项目类型

#### 调增项目（增加应纳税所得额）
- **超标准费用**：业务招待费（按发生额 60% 扣除，最高不超过销售收入 5‰）；广告宣传费（不超过销售收入 15% 部分可扣，超支部分结转）；公益性捐赠（不超过利润总额 12%）。
- **不允许扣除项目**：税收滞纳金、罚款、罚金；赞助支出（非广告性质）；未经核定的准备金支出。
- **关联交易调整**：不满足独立交易原则的关联定价调增。

#### 调减项目（减少应纳税所得额）
- **免税收入**：国债利息收入；符合条件的居民企业之间的股息、红利等权益性投资收益。
- **减计收入**：资源综合利用所得减按 90% 计入收入总额。
- **加计扣除**：研发费用加计扣除（100%）；安置残疾人员工资加计扣除（100%）。

### 1.3 税率适用

| 企业类型 | 税率 | 条件 |
|----------|------|------|
| 一般企业 | 25% | 标准税率 |
| 高新技术企业 | 15% | 经认定 |
| 小型微利企业 | 分段优惠（如 2.5%/5%/10%）| 年应纳税所得额 ≤ 300 万元、从业人数 ≤ 300 人、资产总额 ≤ 5000 万元 |
| 西部大开发企业 | 15% | 鼓励类产业 |
| 软件/集成电路企业 | 两免三减半/10% | 符合认定条件 |

## 2. 应纳税所得额计算步骤

### 步骤 1：取得会计利润
从利润表中获取当期利润总额（税前会计利润）。

### 步骤 2：识别永久性差异
- **永久性差异**：仅影响当期，不产生递延所得税。
- 例如：税收滞纳金（调增）、国债利息收入（调减）。

### 步骤 3：识别暂时性差异
- **暂时性差异**：会在未来期间转回，产生递延所得税。
- 应纳税暂时性差异 → **递延所得税负债**（未来多交税）。
- 可抵扣暂时性差异 → **递延所得税资产**（未来少交税）。

### 步骤 4：计算当期所得税费用
```
当期所得税费用 = (会计利润 ± 永久性差异调整 ± 暂时性差异影响当期部分) × 税率
```

---

## 3. 暂时性差异识别与分类

### 3.1 应纳税暂时性差异（产生递延所得税负债）

| 常见情形 | 说明 |
|----------|------|
| 固定资产加速折旧（税法折旧 > 会计折旧） | 当期少交税，未来多交 |
| 公允价值计量（公允价值 > 计税基础） | 投资性房地产、交易性金融资产 |
| 无形资产资本化（税法加计扣除形成的差异） | 研发支出形成无形资产后产生 |
| 收入确认时间差异 | 税法先确认收入（如预收款方式） |

### 3.2 可抵扣暂时性差异（产生递延所得税资产）

| 常见情形 | 说明 |
|----------|------|
| 资产减值准备 | 会计已计提但税法不允许税前扣除 |
| 预计负债 | 如产品质量保证、未决诉讼 |
| 可结转以后年度的亏损 | 税法允许最长 5 年（高新企业 10 年）内弥补 |
| 递延收益 | 政府补助（与资产相关部分） |
| 超标准广告宣传费结转 | 超过 15% 部分可结转以后年度扣除 |

### 3.3 特殊差异处理

**研发费用加计扣除差异**
- 费用化部分：当期加计扣除 100%，产生永久性差异。
- 资本化部分：形成无形资产后，按 200% 摊销，产生暂时性差异（确认递延所得税）。

## 4. 递延所得税计算与确认

### 4.1 递延所得税资产确认

```
递延所得税资产 = 可抵扣暂时性差异 × 预计转回时的适用税率
```

**确认条件：** 未来很可能（>50%）取得足够的应纳税所得额用于抵扣。

**限制条件：**
- 企业应有确凿证据表明未来期间很可能获得足够应纳税所得额。
- 如无法满足确认条件，不应确认或需计提减值准备（递延所得税资产减值）。

### 4.2 递延所得税负债确认

```
递延所得税负债 = 应纳税暂时性差异 × 预计转回时的适用税率
```

**确认原则：** 除以下特殊情况外，应全额确认：
- 商誉的初始确认
- 企业合并以外的交易中产生的、既不影响会计利润也不影响应纳税所得额的差异

### 4.3 税率变化的处理

当企业所得税税率发生变化时：
1. 重新计量已确认的递延所得税资产和负债。
2. 按新税率调整，差额计入当期所得税费用。
3. 预期税率变动（如高新技术企业资格即将到期）应使用预计的实际税率。

## 5. 所得税费用账务处理

### 5.1 会计分录

```
// 计提当期所得税
借：所得税费用——当期所得税费用
  贷：应交税费——应交企业所得税

// 确认递延所得税资产
借：递延所得税资产
  贷：所得税费用——递延所得税费用

// 确认递延所得税负债
借：所得税费用——递延所得税费用
  贷：递延所得税负债
```

### 5.2 综合分录示例

某企业当期数据如下：
- 会计利润：1000 万元
- 纳税调增：200 万元（含预计负债 100 万元，超标准费用 100 万元）
- 纳税调减：50 万元（国债利息收入）
- 固定资产加速折旧差异（应纳税）：80 万元
- 税率：25%

```
应纳税所得额 = 1000 + 200 - 50 = 1150 万元
当期所得税 = 1150 × 25% = 287.5 万元

递延所得税资产 = 100 × 25% = 25 万元（预计负债可抵扣差异）
递延所得税负债 = 80 × 25% = 20 万元（加速折旧差异）

借：所得税费用——当期所得税费用    287.5 万
  贷：应交税费——应交企业所得税     287.5 万

借：递延所得税资产              25 万
  贷：所得税费用——递延所得税费用    25 万

借：所得税费用——递延所得税费用    20 万
  贷：递延所得税负债              20 万

// 合并：所得税费用 = 287.5 + 20 - 25 = 282.5 万元
```

## 6. 季度预缴与年终汇算清缴

### 6.1 季度预缴
- **预缴方式**：按实际利润额预缴，或按上一纳税年度应纳税所得额平均额预缴。
- **操作**：每季度终了后 15 日内申报预缴企业所得税。
- **会计分录**：
  ```
  借：所得税费用——当期所得税费用
    贷：应交税费——应交企业所得税
  ```

### 6.2 年度汇算清缴
- **时间**：年度终了之日起 5 个月内（次年 1 月 1 日至 5 月 31 日）。
- **步骤**：
  1. 编制年度财务报表。
  2. 进行纳税调整，计算应纳税所得额。
  3. 申报并补缴/退还税款。
  4. 调整递延所得税资产/负债余额。

### 6.3 汇算清缴调整分录
```
// 补缴税款
借：所得税费用——当期所得税费用
  贷：应交税费——应交企业所得税

// 多缴退税
借：应交税费——应交企业所得税
  贷：所得税费用——当期所得税费用
```

## 7. 财务报表列报

### 7.1 资产负债表
- **递延所得税资产**：列报于非流动资产。
- **递延所得税负债**：列报于非流动负债。
- **当期所得税资产/负债**：多缴税款列为流动资产（其他流动资产），应缴税款列为流动负债（应交税费）。

### 7.2 利润表
- 所得税费用 = 当期所得税费用 + 递延所得税费用（或－递延所得税收益）。

### 7.3 附注披露
- 所得税费用的主要构成。
- 当期所得税与会计利润关系的说明（税率调节表）。
- 递延所得税资产/负债的明细及变动。
- 未确认递延所得税资产的可抵扣暂时性差异及可抵扣亏损信息。

## 8. 常见问题与风险提示

1. **亏损弥补期限错误**：一般企业 5 年，高新技术企业/科技型中小企业 10 年。
2. **研发费用加计扣除归集错误**：人工费用直接从事研发活动人员，不包括行政/管理人员；其他相关费用不得超过研发费用总额的 10%。
3. **税率适用错误**：高新技术企业应确保资格有效期内且主营业务符合领域要求。
4. **递延所得税资产确认过度乐观**：无充分证据表明未来有足够应纳税所得额时不应确认。
5. **关联交易定价缺失**：未按独立交易原则进行关联交易申报和调整。
