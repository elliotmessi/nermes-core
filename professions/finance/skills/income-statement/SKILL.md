# 利润表分析

使用本技能时，你需要对企业的**利润表（损益表）进行收入、成本、费用分析，计算毛利率/净利率等关键盈利指标，并进行同比/环比趋势分析**。

---

## 一、数据准备

### 1.1 获取数据
- 所需报表：利润表（损益表）多期数据
- 关键字段：营业收入、营业成本、税金及附加、销售费用、管理费用、研发费用、财务费用、其他收益/损益、营业利润、利润总额、所得税、净利润
- 获取方式：Excel/CSV导入、财务系统API、财报PDF解析

### 1.2 数据预处理
- 确认数据期间（月度、季度、年度）
- 确认可比期间口径一致（同一合并范围、同一会计准则）
- 标记非经常性损益项目（投资收益、资产处置收益、营业外收支等）

---

## 二、利润表结构分析

### 2.1 层次结构
利润表按"多步式"逐层分析：

```
营业收入
- 营业成本
= 毛利润
- 税金及附加
- 销售费用
- 管理费用
- 研发费用
- 财务费用
+ 其他收益/投资净收益/公允价值变动损益
- 信用/资产减值损失
= 营业利润
+ 营业外收入
- 营业外支出
= 利润总额
- 所得税费用
= 净利润
```

### 2.2 垂直分析（共同比分析）
每一项金额占**营业收入**的百分比：
- **毛利率** = 毛利润 / 营业收入 × 100%
- **销售费用率** = 销售费用 / 营业收入 × 100%
- **管理费用率** = 管理费用 / 营业收入 × 100%
- **研发费用率** = 研发费用 / 营业收入 × 100%
- **财务费用率** = 财务费用 / 营业收入 × 100%
- **营业利润率** = 营业利润 / 营业收入 × 100%
- **净利率** = 净利润 / 营业收入 × 100%

### 2.3 费用结构分析
- 期间费用总额占收入比（三费占比）
- 各费用内部明细分析（如销售费用中的广告费、人工费）
- 费用增长率 vs 收入增长率对比（费用弹性分析）

## 三、盈利能力指标计算

### 3.1 核心指标
| 指标 | 计算公式 | 含义 |
|------|---------|------|
| 毛利率 | (收入-成本)/收入 | 产品盈利能力 |
| 营业利润率 | 营业利润/收入 | 经营性盈利水平 |
| 净利率 | 净利润/收入 | 最终盈利水平 |
| EBIT | 净利润+所得税+利息 | 息税前利润 |
| EBITDA | EBIT+折旧+摊销 | 近似经营现金流 |

### 3.2 多维度盈利分析
- **按产品/业务线拆分毛利率** → 识别高/低毛利业务
- **按客户/渠道拆分净利率** → 识别盈利客户群
- **按区域拆分利润率** → 区域市场差异
- **按月度趋势拆分** → 季节性规律

---

## 四、同环比分析

### 4.1 同比分析（Year-over-Year, YoY）
- **同比增长率** = (本期数 - 上年同期数) / 上年同期数 × 100%
- 适用于：消除季节性因素影响
- 重点关注：收入增速、利润增速、毛利率变化

### 4.2 环比分析（Quarter-over-Quarter / Month-over-Month）
- **环比增长率** = (本期数 - 上期数) / 上期数 × 100%
- 适用于：观察短期趋势变化
- 需注意季节性波动对环比的影响

### 4.3 趋势分析表格式
```
| 指标 | 本期 | 上期 | 同比 | 环比 | 上年同期 |
|------|------|------|------|------|---------|
| 营业收入 | 1000 | 900 | +11.1% | +5.0% | 900 |
| 毛利润 | 400 | 350 | +14.3% | +3.0% | 350 |
| 毛利率 | 40% | 38.9% | +1.1pp | -0.8pp | 38.9% |
| 净利润 | 150 | 130 | +15.4% | +2.0% | 130 |
| 净利率 | 15% | 14.4% | +0.6pp | -0.4pp | 14.4% |
```

### 4.4 异常波动识别阈值
- 收入变动率超过 **±30%** → 需核实业务原因
- 毛利率变动超过 **±5个百分点** → 分析成本结构变化
- 期间费用率变动超过 **±3个百分点** → 费用管控异常
- 净利润增速与收入增速**背离超过20%** → 分析成本费用结构

## 五、非经常性损益分析

### 5.1 分类
- **经常性损益**：与主营业务相关的持续性损益
- **非经常性损益**：偶发性、非持续性项目（政府补助、资产处置、重组收益）

### 5.2 扣非净利润
- **扣非净利润** = 净利润 - 非经常性损益
- 扣非净利率 = 扣非净利润 / 营业收入 × 100%
- **核心关注**：如果净利润增长主要来源于非经常性损益，则盈利质量偏低

## 七、使用规则

- 毛利率/净利率计算需注明是**销售毛利率/销售净利率**
- 同环比分析需标注对比期间是否**口径一致**
- 非经常性损益需单独列示，**不得混淆于经常性损益**
- 异常波动需结合业务背景给出**合理解释**，不可只列数字
- 优先使用**扣非净利润**评价经营实质
