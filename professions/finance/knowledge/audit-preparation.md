# 审计准备与配合指南

> 企业应对年度审计和税务稽查的准备清单。提前准备比临时翻箱倒柜省80%的时间。

### 1.2 审计前必做的账务清理

| 序号 | 清理事项 | 说明 |
|------|---------|------|
| 1 | **银行对账** | 所有账户余额调节表差异必须为0 |
| 2 | **应收账款清理** | 长期挂账需确认是否可收回 |
| 3 | **应付账款清理** | 长期未付的无票采购需暂估入账 |
| 4 | **其他应收/应付清理** | 员工借款催收、押金核对、不合理挂账处理 |
| 5 | **存货盘点差异** | 盘盈亏须入账并说明原因 |
| 6 | **固定资产盘点** | 核对台账与实物、补登遗漏资产 |
| 7 | **收入截止测试** | 确保收入确认在正确的会计期间 |
| 8 | **费用截止测试** | 确保跨年费用的正确处理 |
| 9 | **预提费用检查** | 年终奖、年度返利、维修费等须预提 |
| 10 | **税务清理** | 增值税/企业所得税/印花税等各项税款申报完成 |

## 二、审计过程中的配合

### 2.1 审计程序对应准备

| 审计师要做的 | 你需要准备的 |
|------------|------------|
| **函证**（银行+往来） | 提供银行地址/联系人、主要客户/供应商联系方式 |
| **盘点监盘** | 年末存货盘点时通知审计师到场 |
| **收入确认检查** | 大额销售合同+出库单+签收单+发票 |
| **截止测试** | 12月最后几天和1月前几天的大额业务凭据 |
| **关联交易检查** | 关联方名单+关联交易合同+定价说明 |
| **期后事项** | 审计报告日前发生的影响财务报表的重大事项 |

### 2.2 审计沟通技巧

| 场景 | 建议 |
|------|------|
| 审计师发现错报 | 不要急于辩解，先确认事实，然后讨论调整方案 |
| 审计师要求提供资料 | 第一时间响应，不能给的说明原因（别有"拖着就忘了"的侥幸） |
| 审计调整有分歧 | 拿出会计准则依据讨论，不能解决的可升级至事务所技术部 |
| 审计师问敏感问题 | 如实回答，隐瞒的问题最后往往会被其他线索暴出来 |

## 三、税务稽查应对

### 3.1 稽查类型

| 类型 | 触发条件 | 应对要点 |
|------|---------|---------|
| **日常检查** | 例行抽查 | 准备好资料配合检查，态度诚恳 |
| **专项稽查** | 某行业/某税种专项 | 重点关注该税种的合规性 |
| **举报稽查** | 有人举报 | 尽快自查、如确实有问题主动补税减轻处罚 |
| **大数据推送** | 金税四期预警 | 准备解释数电票数据与申报数据差异的原因 |

### 3.2 金税四期常见预警指标

| 预警信号 | 说明 |
|---------|------|
| **进销项不匹配** | 进项税长期大于销项税（长期留抵） |
| **税负率异常** | 增值税税负率远低于同行业水平 |
| **发票顶格开具** | 总是开到发票限额的上限 |
| **大量作废发票** | 当月开次月作废，频繁发生 |
| **进项发票来自"走逃户"** | 上游供应商被列为非正常户 |
| **工资+社保+个税不一致** | 账上工资高、社保基数低、个税申报少 |
| **收入与水电/运费的勾稽异常** | 说收入少但水电费高（制造业） |

### 3.3 稽查现场应对

```
1. 先看《税务检查通知书》+ 检查人员证件
2. 指定专人对接（财务负责人 + 税务专员）
3. 提供资料前先在复印件上注明"此件与原件核对一致"
4. 对于不确定的问题，说"我核实后回复"（不要乱答）
5. 做好书面记录：谁来查、查什么、问了什么问题、你回答了什么
6. 如果稽查结论有误，依法提起行政复议或行政诉讼
```

> **黄金原则**：有问题主动补比被查到罚好。自查补税+滞纳金，比稽查补税+滞纳金+罚款轻得多。

## 四、常见审计调整事项

| 调整事项 | 分录 | 金额判断 |
|---------|------|---------|
| **跨期收入** | 借/贷：主营业务收入 | 影响本年利润的必须调 |
| **跨期费用** | 借/贷：相关费用 | 同上 |
| **资产减值未提** | 借：资产减值损失 贷：减值准备 | 有减值迹象的 |
| **折旧/摊销错误** | 调整累计折旧/摊销 | 计算错误的需更正 |
| **费用跨年混用** | 区分费用化 vs 待摊 | 资本性支出一次性费用化 |
