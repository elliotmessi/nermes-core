# 纳税申报实操手册

> 企业日常纳税申报的完整流程、截止时间、常见错误。适合财务岗日常工作参考。

---

## 一、纳税申报前的准备工作

### 1.1 每月必做清单

| 序号 | 事项 | 说明 |
|------|------|------|
| 1 | **销项数据核对** | 当月开票金额与账面收入核对一致 |
| 2 | **进项发票勾选认证** | 在增值税发票综合服务平台勾选确认 |
| 3 | **收入确认核对** | 无票收入也要申报（不开票≠不交税） |
| 4 | **进项税额转出检查** | 集体福利、个人消费、非正常损失对应的进项税额须转出 |
| 5 | **工资/社保数据** | 与人事部门确认当月工资总额、社保公积金 |
| 6 | **印花税应税凭证统计** | 当月签订的合同、账簿等 |

### 1.2 季度附加工作

| 事项 | 说明 |
|------|------|
| **企业所得税预缴** | 按季度利润预缴所得税 |
| **财务报表上传** | 季度末上传资产负债表、利润表、现金流量表 |

## 二、增值税申报

### 2.1 一般纳税人申报流程

```
① 进项发票勾选认证（月底前完成）
② 抄报税（月初 → 开票系统自动汇总上月开票数据）
③ 填写增值税申报表（电子税务局）
④ 核对应纳税额
⑤ 申报并缴款（次月15日前）
```

### 2.2 增值税申报表核心行次

| 行次 | 内容 | 说明 |
|------|------|------|
| 第1行 | 按适用税率计税销售额 | = 专票销售额 + 普票销售额 + 无票收入 |
| 第11行 | 销项税额 | = 第1行 × 税率 |
| 第12行 | 进项税额 | 当期已认证的进项税额 |
| 第17行 | 应抵扣税额合计 | = 12+13-14-15+16 |
| 第19行 | 应纳税额 | = 11-18 |
| 第24行 | 应纳税额合计 | = 19+21-23 |
| 第34行 | 本期应补（退）税额 | 最终应缴金额 |

> **常见错误**：忘记填报无票收入。账外的现金收款、微信/支付宝收款、个人转账的收入都要申报。

### 2.3 小规模纳税人申报

- 按季度申报
- 月销售额 ≤ 10万元（季度 ≤ 30万元）：免征增值税
- 超过免税额度：按 1% 征收率（2026年延续）全额纳税
- 申报表比一般纳税人简单很多

## 三、企业所得税预缴

### 3.1 预缴方式

| 方式 | 适用 | 计算公式 |
|------|------|---------|
| **据实预缴**（最常用） | 多数企业 | 实际利润额 × 税率 |
| **按上年平均额** | 税务机关核定 | 上年度应纳税额 / 12（或4）× 月数 |

### 3.2 预缴申报表要点

| 行次 | 内容 |
|------|------|
| 第1行 | 营业收入（1-本季累计） |
| 第2行 | 营业成本 |
| 第3行 | 利润总额（注意：不是净利润） |
| 第9行 | 实际利润额 = 3+4-5-6-7-8 |
| 第12行 | 应纳所得税额 = 9 × 税率 |
| 第13行 | 减免所得税额（小微/高新等优惠） |
| 第15行 | 本期应补（退）所得税额 |

> **关键**：预缴时研发费用加计扣除、固定资产加速折旧等优惠政策可在预缴时享受，不需等汇算清缴。

## 四、个人所得税代扣代缴

### 4.1 工资薪金个税计算

```
应纳税所得额 = 累计收入 - 累计减除费用(5000×月数) - 累计专项扣除 - 累计专项附加扣除 - 累计其他扣除
应纳税额 = 应纳税所得额 × 税率 - 速算扣除数 - 累计已预缴税额
```

### 4.2 申报操作

- 使用"自然人电子税务局（扣缴端）"或在电子税务局申报
- 每月15日前完成上月个税申报
- 人员增减、专项附加扣除信息需及时更新

### 4.3 常见风险点

| 风险 | 说明 |
|------|------|
| 两处以上工资 | 各支付单位分别扣缴，年度汇算合并计算 |
| 劳务报酬未代扣 | 支付给个人的劳务费（含专家咨询费）须代扣20%-40% |
| 年终奖单独计税 | 每人每年只能用一次 |

## 五、印花税申报

### 5.1 申报周期

- 按季度申报（多数地区）
- 或按次申报（业务量少的企业）

### 5.2 应税合同印花税计算

| 合同类型 | 税率 | 计税依据 |
|---------|------|---------|
| 买卖合同 | 0.3‰ | 合同金额 |
| 借款合同 | 0.05‰ | 借款金额 |
| 租赁合同 | 1‰ | 租金总额 |
| 技术合同 | 0.3‰ | 合同金额 |

> 小微企业和个体工商户印花税减半（至2027年12月31日）。

## 六、汇算清缴要点（年度）

### 6.1 企业所得税汇算清缴

**时间**：次年1月1日—5月31日

**必须完成的调整事项**：

| 调整项目 | 常见情况 |
|---------|---------|
| **业务招待费** | 按发生额60%扣除，上限为收入的5‰。超出部分纳税调增 |
| **广告费** | 一般企业不超过收入15%。超出部分结转以后年度扣除 |
| **公益性捐赠** | 不超过利润总额12%。超出部分结转3年 |
| **职工福利费** | 不超过工资总额14% |
| **工会经费** | 不超过工资总额2% |
| **职工教育经费** | 不超过工资总额8% |
| **资产减值准备** | 未经核准的准备金不得税前扣除，纳税调增 |
| **行政罚款/滞纳金** | 不得税前扣除，全额调增 |

### 6.2 个人所得税汇算清缴

**时间**：次年3月1日—6月30日

**需要汇算的情形**：
- 已预缴税额 > 年度应纳税额（可申请退税）
- 年度综合所得收入 > 12万元且补税 > 400元
- 有两处以上工资薪金收入

## 七、申报常见错误TOP 5

| 排名 | 错误 | 后果 | 预防 |
|------|------|------|------|
| 1 | **无票收入漏申报** | 税务稽查时发现账外收入，补税+罚款 | 建立无票收入台账，每月与账面核对 |
| 2 | **进项税额应转出未转出** | 虚增进项抵扣，补税+滞纳金 | 每月末检查是否有集体福利/个人消费对应的进项 |
| 3 | **发票跨年入账** | 汇算清缴调整麻烦 | 年底催收未到发票，暂估入账 |
| 4 | **股权转让未报税** | 被税务局大数据比对发现 | 股东变更及时做税务申报 |
| 5 | **关联交易未申报** | 不符合独立交易原则的补税 | 关联交易需准备同期资料 |
