# 成本核算方法实务

> 制造业、服务业常用的成本核算方法及适用场景。选择合适的核算方法直接影响产品定价和利润分析。

---

## 一、成本核算基础

### 1.1 成本分类

```
总成本 = 直接材料 + 直接人工 + 制造费用
       = 变动成本 + 固定成本
```

| 类型 | 定义 | 举例 |
|------|------|------|
| **直接材料** | 直接构成产品实体的原材料 | 汽车制造中的钢材、玻璃、轮胎 |
| **直接人工** | 直接参与生产的一线工人工资 | 车间操作工工资 |
| **制造费用** | 间接费用，需分摊 | 车间折旧、水电、车间管理人员工资 |
| **变动成本** | 随产量同比例变化的成本 | 原材料、计件工资 |
| **固定成本** | 不随产量变化的成本 | 厂房租金、设备折旧 |

### 1.2 本量利分析（盈亏平衡）

```
盈亏平衡点（销量）= 固定成本 / (单价 - 单位变动成本)
                  = 固定成本 / 单位边际贡献
```

```
盈亏平衡点（销售额）= 固定成本 / 边际贡献率
边际贡献率 = (单价 - 单位变动成本) / 单价 × 100%
```

> 例：固定成本 100 万/年，单价 200 元，单位变动成本 120 元
> 盈亏平衡 = 100万 / (200-120) = 12,500 件
> 超过12,500件后才开始盈利。

## 二、三种核心核算方法

### 2.1 品种法 — 按产品品种归集

**适用**：大批量单步骤生产（如发电厂、采掘、铸造）

```
某产品成本 = 直接材料 + 直接人工 + 该产品分配到的制造费用
单位成本 = 某产品总成本 / 该产品当月产量
```

**特点**：
- ✅ 最简单，计算量小
- ❌ 不能反映各工序的成本，无法做车间考核

### 2.2 分批法（订单法）— 按生产批次归集

**适用**：单件小批量、按订单生产（如造船、模具、大型设备）

```
某订单成本 = 该订单累计投入的直接材料 + 直接人工 + 分摊的制造费用
```

**特点**：
- ✅ 精确反映每个订单的成本
- ❌ 月末在制品多，跨月订单的成本计算复杂

### 2.3 分步法 — 按生产步骤归集

**适用**：多步骤连续加工（如纺织、化工、食品、电子组装）

```
步骤1成本 → 转入步骤2 → 步骤2成本 → ... → 产成品成本
（逐步结转）
```

**特点**：
- ✅ 反映各步骤成本，可做车间考核
- ❌ 核算工作量大，需用约当产量计算在制品

| 分步法变种 | 原理 |
|-----------|------|
| **逐步结转分步法** | 每一步的成本实物和价值一起向后传 |
| **平行结转分步法** | 各步骤只算自己的加工费，最终汇总 |

## 三、制造费用分摊

### 3.1 传统分摊法

```
分配率 = 制造费用总额 / 分配标准总量
某产品应摊制造费用 = 该产品分配标准用量 × 分配率
```

**常见分配标准**：
| 标准 | 适用场景 |
|------|---------|
| 直接人工工时 | 劳动密集型 |
| 机器工时 | 自动化程度高 |
| 直接人工成本 | 工资水平差异大时 |
| 直接材料成本 | 材料成本占比大的重工业 |

### 3.2 作业成本法（ABC）

对于产品线多、间接费用高的企业，传统分摊法可能严重扭曲成本。

ABC 法的思路：
```
制造费用 → 按作业动因归集到各作业中心 → 按成本动因分摊到各个产品
```

| 作业中心 | 成本动因 | 举例 |
|---------|---------|------|
| 采购作业 | 订单次数 | 小批量多品种采购的隐形成本 |
| 质检作业 | 检验批次 | 高精度产品的质检成本 |
| 设备调试 | 调试次数 | 小批量生产频繁换线的成本 |
| 物料搬运 | 搬运次数/距离 | 仓库布局不好的搬运成本 |

> 传统分摊：小批量高毛利产品可能"看起来"很赚钱。
> ABC 法后：发现它占用了大量换线、调试、质检资源，其实是亏的。

## 四、在制品（WIP）成本计算

### 4.1 约当产量法

月末有未完工的在制品时，需将其折算为"相当于多少件完工产品"。

```
某成本项目的约当产量 = 当月完工产量 + 月末在制品约当量
月末在制品约当量 = 月末在制品数量 × 完工程度

某成本项目分配率 = (月初在制品成本 + 本月发生成本) / 约当产量
完工产品应负担 = 完工产量 × 分配率
月末在制品应负担 = 在制品约当量 × 分配率
```

**完工程度判定**：

| 成本项目 | 投料方式 | 完工程度判定 |
|---------|---------|------------|
| 直接材料 | 开工一次投料 | 在制品按100%计算 |
| 直接材料 | 陆续投料 | 在制品按实际完工程度 |
| 直接人工 | — | 按实际完工程度（通常50%） |
| 制造费用 | — | 按实际完工程度（通常50%） |

### 4.2 简易替代法

小企业在制品极少时可用：
- 在制品按原材料成本计价
- 或在制品不计算（当月全部计入完工产品成本）

## 五、成本控制工具

### 5.1 标准成本法

```
标准成本 = 标准用量 × 标准价格

差异分析：
  用量差异 = (实际用量 - 标准用量) × 标准价格
  价格差异 = (实际价格 - 标准价格) × 实际用量
```

每月分析差异 → 找出超支原因 → 改进

### 5.2 成本降低的常见路径

| 路径 | 方法 |
|------|------|
| 降低采购成本 | 集中采购、年度框架协议、替代材料 |
| 提高材料利用率 | 优化下料方案、减少废品率 |
| 减少浪费 | 精益生产、5S管理 |
| 提高效率 | 自动化设备、减少换线时间 |
| 外包非核心环节 | 比自己做更便宜的部分外包 |
