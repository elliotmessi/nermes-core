# 薪资处理技能

## 技能描述

薪资处理技能用于企业员工工资的计算、审核与发放全流程管理。系统根据员工的考勤数据、绩效考核结果、社保公积金缴纳基数、个税专项附加扣除等信息，自动计算应发工资、代扣代缴项目和实发工资。支持多种薪资结构（月薪制、日薪制、计件制、提成制等），并与银行代发系统对接实现批量发放。

## 适用场景

- **月度工资核算**：每月根据考勤和绩效数据批量计算员工工资，生成工资条
- **年终奖计算**：按全年一次性奖金政策计算年终奖个税，优化税负
- **社保公积金代扣**：按当地社保局和公积金管理中心要求自动计算应缴金额
- **个税累计预扣**：按照新个税法累计预扣法计算每月应预扣预缴个税

## 输入格式

```json
{
  "payroll_period": "2026-05",
  "employees": [
    {
      "employee_id": "EMP001",
      "name": "张三",
      "basic_salary": 15000.00,
      "attendance_days": 21,
      "overtime_hours": 8,
      "performance_bonus": 3000.00,
      "social_security_base": 15000.00,
      "housing_fund_base": 15000.00,
      "special_deductions": [
        {"item": "子女教育", "amount": 2000.00},
        {"item": "住房贷款利息", "amount": 1000.00}
      ]
    }
  ]
}
```

## 注意事项

- 社保公积金基数每年调整一次，需在各地规定时间窗口内完成基数申报
- 新入职员工应按实际入职月份计算减除费用，避免个税多扣或少扣
- 加班费计算需遵守劳动法规定，工作日、休息日和法定节假日的加班倍率不同
- 薪资数据属于高度敏感信息，系统需设置严格的权限控制和操作日志
- 工资发放后需及时打印工资表，由员工签字确认并存档备查
