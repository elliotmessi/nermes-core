# 应收账款管理技能

## 技能描述

应收账款管理技能用于企业对客户销售款项的全流程管理，包括销售订单确认、发票开具、收款跟踪、账龄分析和催收管理等功能。系统自动记录每笔应收账款的产生、回款和核销过程，通过信用评级模型评估客户信用风险，设置预警阈值。支持智能催收计划生成、坏账准备计提测算和应收账款证券化可行性分析。

## 适用场景

- **销售回款跟踪**：实时追踪各客户的到期应收款项，自动提醒逾期款项
- **客户信用管理**：根据客户的付款历史、财务状况等评估信用额度并动态调整
- **催收流程管理**：按照逾期天数和客户等级，自动生成催收行动计划（短信、邮件、电话、法务）
- **应收账龄分析与坏账评估**：定期分析应收账款的账龄结构，评估坏账风险并计提准备

## 注意事项

- 应收账款周转天数（DSO）是衡量回款效率的核心指标，应持续跟踪并与行业水平对标
- 新客户应在发货前完成信用评估，大额订单可要求预付定金或银行保函
- 催收过程需保留完整的沟通记录（电话录音、邮件截图、催收函扫描件）作为法律证据
- 超过3年的应收款项在税务上已无法税前扣除坏账损失，需及时处理
- 应收账款的核销需经审批，严禁私自核销客户欠款
