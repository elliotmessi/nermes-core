# 发票管理技能

本技能提供发票管理全流程指导，涵盖全电发票（数电票）的开具与处理、红字发票操作、发票认证勾选、税控设备使用及常见问题处理。

---

## 1. 发票类型概览

### 1.1 按形式分类

| 发票类型 | 说明 | 主要特征 |
|----------|------|----------|
| 纸质发票 | 传统纸质增值税发票 | 需使用税控盘开具，需领用纸质空白发票 |
| 电子发票（普通） | 增值税电子普通发票 | OFD 格式，全国统一的电子发票服务平台 |
| 全电发票（数电票） | 全面数字化电子发票 | 无需税控设备，全程线上，2022 年起全面推广 |
| 区块链发票 | 基于区块链技术的电子发票 | 部分地区试点 |

### 1.2 按用途分类

| 发票类型 | 用途 | 对方能否抵扣 |
|----------|------|-------------|
| 增值税专用发票 | 一般纳税人销售货物/服务 | 可抵扣进项税 |
| 增值税普通发票 | 所有纳税人销售货物/服务 | 不可抵扣（除特定情形）|
| 机动车销售统一发票 | 机动车销售专用 | 符合条件可抵扣 |
| 二手车销售统一发票 | 二手车交易专用 | 不可抵扣 |
| 通行费电子发票 | 道路通行服务 | 上端注明税额可抵扣 |
| 增值税电子普通发票（旅客运输）| 国内旅客运输服务 | 注明旅客身份信息可计算抵扣 |

## 2. 全电发票（数电票）操作指南

### 2.1 什么是全电发票

全电发票（全面数字化电子发票）是国家税务总局推广的新型发票形式，主要特点：
- **无需税控设备**：通过电子税务局直接开具
- **无最大开票额度限制**：额度由系统自动核定
- **信息实时上传**：开具即上传国家税务总局数据库
- **无需领用空白发票**：额度内自行开具
- **统一格式**：XML/PDF/OFD 多格式下载

### 2.2 全电发票开票流程

#### 步骤 1：登录电子税务局
登录各省电子税务局 -> 进入"税务数字账户"或"全电发票"模块。

#### 步骤 2：开具蓝字发票
1. 进入"开票业务" -> "蓝字发票开具"。
2. 填写购买方信息（名称、纳税人识别号、地址电话、开户行及账号）。
3. 填写商品/服务信息（项目名称、数量、单价、金额、税率）。
4. 选择"是否含税"，系统自动计算合计金额和税额。
5. 确认无误后点击"开具"。
6. 系统自动生成发票号码和发票二维码。
7. 可通过邮件、短信等方式发送给购买方。

#### 步骤 3：查看与交付
1. 在"发票查询"模块查看已开具发票清单。
2. 下载 PDF/OFD/XML 格式文件。
3. 通过"发票交付"功能将电子发票发送给客户。

### 2.3 全电发票额度管理

- **授信额度**：税务机关根据纳税人经营情况自动核定每月（或每次）最高开票额度。
- **额度调整**：如实际业务需求超出授信额度，可在线提交"发票额度调整申请"。
- **额度查询**：在税务数字账户 -> 发票额度管理 -> 发票额度查询中查看。

### 2.4 全电发票接收与查验

1. **接收渠道**：邮件、短信链接、企业财务系统接口等。
2. **查验渠道**：
   - 国家税务总局全国增值税发票查验平台（inv-veri.chinatax.gov.cn）
   - 电子税务局"发票查验"模块
   - 第三方财税平台查验
3. **归档要求**：接收方以 PDF/OFD 电子文件形式存档，无需打印纸质件。

## 3. 红字发票操作

### 3.1 红字发票开具情形

| 情形 | 说明 |
|------|------|
| 销售退回 | 已确认收入的商品发生退货 |
| 销售折让 | 因质量问题给予价格折扣 |
| 开票有误 | 发票内容填错（金额、税率、购买方等）|
| 服务中止 | 服务合同提前终止 |
| 折扣折让 | 事后给予的商业折扣 |

### 3.2 红字发票开具流程

#### 专票红冲流程

**购买方已认证抵扣：**
1. 购买方在税务数字账户申请《红字信息确认单》（原增值税发票管理系统为《红字发票信息表》）。
2. 销售方接收确认单后，开具红字发票。
3. 销售方需以原蓝字发票为基础，填写负数金额和税额。

**购买方未认证抵扣/发票未交付：**
1. 销售方在税务数字账户申请《红字信息确认单》并直接开具红字发票。
2. 开票时选择原蓝字发票信息，系统自动生成负数数据。

#### 普票红冲流程
1. 销售方直接在税控系统或电子税务局中开具红字普通发票。
2. 必须关联对应的原蓝字发票或注明原发票号码代码。
3. 全电发票普票红冲直接在"红字发票开具"模块操作。

### 3.3 全电发票红冲操作步骤
1. 进入"红字发票开具"模块。
2. 选择原蓝字发票（系统会自动带出原发票信息）。
3. 选择红冲原因（销售退回/开票有误/销售折让/服务中止）。
4. 系统生成红字发票确认单（如需购买方确认）。
5. 确认后开具红字发票。
6. 红字发票号码自动生成，信息同步至国家税务总局数据库。

### 3.4 红字发票注意事项
- 红字发票必须在原蓝字发票开具之日起 360 日内开具（全电发票同步执行）。
- 红字发票的税率、商品名称须与原蓝字发票一致。
- 红字发票不能单独申请抵扣（购买方需做进项税额转出）。
- 跨月的红字发票无法作废，只能再次红冲。

## 4. 发票认证勾选

### 4.1 认证方式演变

| 阶段 | 方式 | 操作 |
|------|------|------|
| 传统方式 | 扫描认证 | 扫描仪扫描纸质发票抵扣联 |
| 过渡期 | 网上认证 | 登录增值税发票选择确认平台 |
| 当前主流 | 综合服务平台 | 增值税发票综合服务平台（勾选确认）|
| 最新方式 | 税务数字账户 | 全电发票时代，集成在电子税务局内 |

### 4.2 勾选认证操作流程（增值税发票综合服务平台）

1. **登录**：使用税控设备（税控盘/UKey）登录增值税发票综合服务平台。
2. **发票抵扣勾选**：
   - 进入"抵扣勾选" -> "发票抵扣勾选"。
   - 输入查询条件（发票代码号码、开票日期范围）查询发票。
   - 勾选需要认证抵扣的发票。
   - 确认勾选（系统会统计已勾选发票的金额和税额）。
3. **申请统计**：勾选完成后，进入"抵扣勾选统计"页面，点击"申请统计"。
4. **确认签名**：统计完成后，点击"确认"完成签名确认。
5. **下载抵扣明细**：下载 Excel/PDF 格式的抵扣明细留存备查。

### 4.3 税务数字账户勾选（全电发票）

1. 登录电子税务局 -> 进入"税务数字账户"。
2. 选择"发票勾选确认" -> "抵扣类勾选"。
3. 系统自动展示可勾选发票清单，也可手动输入发票号码查询。
4. 勾选需要认证的发票，确认后提交。
5. 完成统计确认。

### 4.4 认证时限

- **传统专票**：已取消 360 日认证期限限制（2020 年 3 月 1 日起），但建议在申报期前完成认证。
- **全电发票**：实时上传，勾选认证无时间限制，但应在纳税申报前完成。
- **海关专用缴款书**：需在电子税务局-海关缴款书模块进行稽核比对，通过后方可抵扣。

### 4.5 认证异常处理

- **发票查无此票**：联系开票方确认发票是否已上传国家数据库。
- **发票已认证/已使用**：确认是否被其他纳税人错误抵扣。
- **发票信息不符**：销售方品名/金额/税率错误，联系对方重开。
- **发票状态异常**：如为失控发票、异常凭证，需做进项税额转出。

## 5. 税务UKey/税控盘操作

### 5.1 税控设备对比

| 设备类型 | 适用对象 | 特点 |
|----------|----------|------|
| 税务UKey | 新办纳税人/小规模 | 免费，即办即领，USB 接口 |
| 税控盘 | 一般纳税人（存量） | 原防伪税控系统继续使用 |
| 金税盘 | 原税控系统 | 逐步淘汰中 |

### 5.2 日常操作

**发票领用（税控盘）：**
1. 电子税务局 -> 发票使用 -> 发票领用申请。
2. 选择发票种类和数量，提交申请。
3. 选择领取方式（邮寄/上门领取）。
4. 领用后登录税控软件 -> 发票管理 -> 发票读入。

**已开发票查询：**
1. 税控软件 -> 发票管理 -> 已开发票查询。
2. 按月份/季度查询已开发票列表。
3. 支持汇总打印和导出。

**发票作废：**
1. 当月开具且未跨月的发票可作废。
2. 税控软件 -> 发票管理 -> 发票作废。
3. 选择需要作废的发票编号，确认作废。
4. 已跨月的发票只能红冲不能作废。

### 5.3 年度数据备份与迁移

1. 每年年末进行一次税控设备数据备份。
2. 备份路径：税控软件 -> 数据管理 -> 数据备份。
3. 更换设备时需进行"税控设备变更发行"和"数据迁移"。
4. 企业注销时需办理税控设备注销发行。

## 6. 发票涉税风险防范

### 6.1 发票合规要点

- **三流一致**：资金流（银行收付）、发票流（发票开具）、货物流（合同、物流单据）保持一致。
- **真实交易**：不得开具与实际经营不符的发票（虚开）。
- **准确性**：发票内容（品名、数量、金额、税率）与合同一致。
- **及时性**：在纳税义务发生时间开具发票，不得提前或滞后。

### 6.2 异常凭证处理

收到"异常增值税扣税凭证"通知时：
1. 确认是否为异常凭证（系统自动推送）。
2. 如涉及出口退税，暂不办理退税。
3. 在 10 个工作日内向税务机关提供证明材料（合同、付款凭证、物流单据、出入库单等）。
4. 如确属真实交易，税务机关核实后可转为正常凭证。
5. 如无法提供有效证明，需做进项税额转出处理。

### 6.3 虚开发票防范

- 核实购买方纳税识别号的真实性。
- 审核交易合同和资金往来的真实性。
- 不参与"代开发票""买发票"等违法活动。
- 对大额发票交易进行事前、事中、事后全程跟踪。

## 7. 常见问题解答

**Q：全电发票和电子发票的区别？**
A：全电发票是全面数字化的电子发票，无需税控设备，无需领用空白发票，直接通过电子税务局开具；传统电子发票仍需税控设备和领用发票。

**Q：发票查验平台提示查无此票？**
A：可能原因：发票尚未上传国家数据库（通常开票后 24-48 小时同步）。建议等待 48 小时后重新查询。若仍查不到，联系开票方确认。

**Q：认证后发现有误怎么办？**
A：当月未申报前，可在勾选平台撤销勾选；已申报的，需在申报后进行进项税额转出。

**Q：全电发票可以作废吗？**
A：全电发票不能作废，只能红冲。开具后发现错误的，直接开具红字发票。

**Q：红字发票信息表如何撤销？**
A：未使用的红字发票信息表可直接在系统撤销；已使用的需联系税务机关处理。
