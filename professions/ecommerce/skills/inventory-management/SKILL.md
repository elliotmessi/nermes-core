# 库存管理技能 — 安全库存与补货预警

## 功能描述
帮助电商运营者建立科学的库存管理体系，包括安全库存计算、补货预警、库存周转分析、滞销品处理建议，降低缺货和积压风险。

## 核心能力

### 1. 安全库存计算
- **公式模型**：安全库存 = Z值（服务水平系数）× √(LTσD² + D²σLT²)，其中 LT=提前期，D=日均需求
- **简化模型**：安全库存 = 日均销量 × 补货天数 × 1.5（适用于中小卖家）
- **季节性调整**：大促期间安全库存系数提高到2-3倍

### 2. 补货预警
- **再订货点**：再订货点 = 日均销量 × 补货天数 + 安全库存
- **预警等级**：
  - 🔴 红色预警：库存 ≤ 3天销量，立即补货
  - 🟡 黄色预警：库存 ≤ 7天销量，准备下单
  - 🟢 绿色：库存充足
- **自动预警**：基于ERP数据设置低库存自动通知

### 3. 库存周转分析
- **库存周转率**：销售成本 / 平均库存余额
- **周转天数**：30天 / 月周转率
- **健康标准**：服装鞋包 30-60天，标品/快消品 15-30天，大家电 45-90天
- **优化方向**：提升周转 = 减少资金占用，降低仓储成本

### 4. 滞销品管理
- **滞销定义**：30天以上0销量或月销≤1件
- **处理方案**：
  - 清仓折扣：梯度降价
  - 捆绑销售：搭配热销品
  - 赠品消耗：作为活动赠品
  - 退货给供应商（如有约定）

### 5. 多仓库存
- 京东入仓 / 菜仓 / 拼多多云仓
- 多仓库存分配策略
- 预售库存管理

## 使用场景
- 新品上架后的首批补货计算
- 大促前备货量评估
- 月度库存盘点与优化
- 滞销品清理方案设计

## 输入
商品历史销售数据（日均销量、波动系数）、供应商补货周期、当前库存量、在途库存

## 输出
安全库存建议值、补货时间点推荐、库存健康度评分、滞销品处理方案
