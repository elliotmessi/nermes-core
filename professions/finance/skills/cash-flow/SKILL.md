# 现金流量表分析

使用本技能时，你需要对企业的**现金流量表进行编制（直接法/间接法）、三大活动（经营/投资/筹资）分析，并评估现金流质量**。

---

## 一、现金流量表结构

### 1.1 三大现金流量活动

```
期初现金及现金等价物余额
+ 经营活动产生的现金流量净额     → 核心造血能力
+ 投资活动产生的现金流量净额     → 资本支出/投资回收
+ 筹资活动产生的现金流量净额     → 融资/偿债/分红
+ 汇率变动影响
= 期末现金及现金等价物余额
```

### 1.2 现金流特点判断
| 经营现金流 | 投资现金流 | 筹资现金流 | 典型阶段 |
|-----------|-----------|-----------|---------|
| +（正） | -（负） | +（正） | 成长期：业务造血且投资扩张，外部融资 |
| +（正） | -（负） | -（负） | 成熟期：业务造血支持投资，同时偿还融资 |
| +（正） | +（正） | -（负） | 衰退期：缩减投资同时偿债/分红 |
| -（负） | -（负） | +（正） | 初创/困境期：无造血能力，依赖外部输血 |
| -（负） | +（正） | +（正） | 危险信号：主业亏损靠卖资产和融资维持 |

## 二、直接法编制与调整

### 2.1 直接法逻辑
直接法按现金收支的实际发生额列示，适用于编制**管理用现金流量表**：

```
销售商品、提供劳务收到的现金
- 购买商品、接受劳务支付的现金
- 支付给职工以及为职工支付的现金
- 支付的各项税费
- 支付的其他与经营活动有关的现金
= 经营活动现金净额
```

### 2.2 直接从账套获取
如果财务系统支持，直接从**现金日记账/银行对账单**按业务性质分类汇总：
- 收入类：按客户回款流水归集
- 支出类：按供应商付款、工资、税费等分类归集

## 三、间接法编制与调整

### 3.1 间接法转换逻辑
从净利润出发，调整非现金项目和非经营活动损益：

```
净利润
+ 资产减值损失 / 信用减值损失       → 非现金支出，调增
+ 折旧与摊销                        → 非现金支出，调增
+ 财务费用（利息支出）               → 筹资活动，调增
- 投资收益                          → 投资活动，调减
+ 公允价值变动损失                   → 非现金，调增
+ 存货减少                          → 资产减少占用的现金释放
- 存货增加                          → 现金被存货占用，调减
+ 经营性应收项目减少                  → 回款增加，调增
- 经营性应收项目增加                  → 资金被占用，调减
+ 经营性应付项目增加                  → 占用供应商资金，调增
- 经营性应付项目减少                  → 偿还欠款，调减
= 经营活动产生的现金流量净额
```

### 3.2 调整要点
- **经营性应收**：应收账款、应收票据（经营性部分）、预付款项、合同资产等
- **经营性应付**：应付账款、应付票据、合同负债、应付职工薪酬、应交税费等
- **非经营性应收/应付**（如与投资/筹资相关的）**不得计入**
- 利息收入/支出、股利收入 → 归入对应活动，需从经营中剔除

## 四、经营活动现金流分析

### 4.1 核心指标
- **经营活动现金流量净额** > 0 → 企业具备造血能力
- **经营现金净额 vs 净利润** → 盈利的"含金量"
  - 连续多期经营现金净额 < 净利润 → 利润"虚胖"，大量应收账款
  - 经营现金净额 > 净利润 → 回款良好或折旧摊销等非现金支出较多

### 4.2 现金流质量评价
- **销售收现率** = 销售商品收到的现金 / 营业收入 × 100%
  - > 100% → 回款良好，既有本期回款也有前期欠款收回
  - < 100% → 存在赊销，应收账款增加
- **现金利润比** = 经营现金流量净额 / 净利润
  - 长期 > 1 → 盈利质量优秀
  - 长期 < 0.5 → 盈利质量堪忧

### 4.3 自由现金流（FCF）
- **自由现金流 = 经营现金流量净额 - 资本性支出**
- 自由现金流 > 0 → 企业有富余资金用于偿债、分红或再投资

## 五、投资活动现金流分析

### 5.1 支出端
- **构建固定资产/无形资产支付的现金** → 扩张性投资（好）还是维持性投资？
- **投资支付的现金** → 对外股权投资或金融资产投资
- **取得子公司支付的现金** → 并购活动

### 5.2 回收端
- **收回投资收到的现金** → 资产处置或投资到期
- **取得投资收益收到的现金** → 分红、利息等

### 5.3 判断标准
- 持续大额投资支出 + 经营现金流为正 → 积极扩张
- 投资支出萎缩或依赖处置资产获取现金 → 收缩战略或资金链紧张

---

## 六、筹资活动现金流分析

### 6.1 流入端
- **吸收投资收到的现金** → 股权融资
- **取得借款收到的现金** → 债务融资

### 6.2 流出端
- **偿还债务支付的现金** → 债务到期偿付
- **分配股利、利润或偿付利息支付的现金** → 股东回报及融资成本
- **回购股份支付的现金** → 股份回购

### 6.3 偿债能力判断
- **现金利息保障倍数** = (经营现金净额 + 利息支出 + 所得税) / 利息支出
  - > 2 → 安全
  - < 1 → 偿债风险高
- **债务偿还覆盖率** = 经营现金净额 / (短期借款+一年内到期长期借款)
  - > 1 → 短期偿债无忧

## 八、使用规则

- **直接法**用于管理分析（内部决策），**间接法**用于对外报告（会计准则）
- 间接法调整项需确保**不与净利润中的非经营活动损益重复**调整
- 自由现金流计算需明确是否包含**维护性资本支出 **或**全部资本支出**
- 分不清现金流入/流出性质时，参考会计科目对应的**业务实质**而非科目名称
- 跨期汇率变动影响需单独列示，不得混入三大活动
