# 财务比率分析

使用本技能时，你需要对企业的**财务报表进行计算四大类财务比率（偿债能力、营运能力、盈利能力、成长能力），并结合行业基准进行解读与评价**。

---

## 一、数据准备

### 1.1 所需数据源
- 资产负债表（期末/期初余额）
- 利润表（本期/上年同期）
- 现金流量表（本期/上年同期）
- **行业参考值**：同行业上市公司均值/中位数、行业白皮书、权威数据库

### 1.2 计算前提
- **双期数据**：比率中涉及存量（时点数）与流量（时期数）时，存量用**平均数**
  - 平均总资产 = (期初总资产 + 期末总资产) / 2
  - 平均净资产 = (期初净资产 + 期末净资产) / 2
  - 平均应收账款 = (期初应收账款 + 期末应收账款) / 2
- 特殊行业需调整标准公式（如金融行业、房地产行业）

---

## 二、偿债能力分析

### 2.1 短期偿债能力

| 指标 | 公式 | 参考值 | 含义 |
|------|------|--------|------|
| 流动比率 | 流动资产 / 流动负债 | 1.5~2.0 | 1元流动负债有多少流动资产保障 |
| 速动比率 | (流动资产-存货) / 流动负债 | 0.8~1.2 | 去除存货后的偿债能力 |
| 现金比率 | (货币资金+交易性金融资产) / 流动负债 | 0.2~0.5 | 最直接的支付能力 |
| 营运资金 | 流动资产 - 流动负债 | > 0 | 可用于日常经营的资金 |

### 2.2 长期偿债能力

| 指标 | 公式 | 参考值 | 含义 |
|------|------|--------|------|
| 资产负债率 | 总负债 / 总资产 × 100% | 40%~60% | 杠杆水平 |
| 权益乘数 | 总资产 / 净资产 | 1.5~3.0 | 财务杠杆倍数 |
| 利息保障倍数 | EBIT / 利息费用 | > 2.0 | 利润覆盖利息的能力 |
| 产权比率 | 总负债 / 净资产 | 0.5~1.5 | 负债与所有者权益的比例 |

### 2.3 分析要点
- 流动比率高 ≠ 绝对安全：需检查流动资产质量（如大量积压存货或长账龄应收）
- 资产负债率需**结合行业判断**：房地产/金融行业通常高于制造业
- 利息保障倍数 < 1 时企业已无法通过利润支付利息（危险信号）

## 三、营运能力分析

| 指标 | 公式 | 含义 |
|------|------|------|
| 应收账款周转率(次) | 营业收入 / 平均应收账款 | 越高越好→回款快 |
| 应收账款周转天数 | 365 / 周转率 | 回款周期 |
| 存货周转率(次) | 营业成本 / 平均存货 | 越高越好→销售快 |
| 存货周转天数 | 365 / 周转率 | 存货持有周期 |
| 总资产周转率(次) | 营业收入 / 平均总资产 | 资产使用效率 |
| 固定资产周转率(次) | 营业收入 / 平均固定资产 | 固定资产产出效率 |
| 应付账款周转天数 | 平均应付账款 / 营业成本 × 365 | 占用供应商资金周期 |

### 3.1 营运资金周期
- **现金转化周期** = 存货周转天数 + 应收账款周转天数 - 应付账款周转天数
- 正天数：资金被占用，需要营运资金支持
- 负天数：占用上下游资金（如零售行业）
- 越短越好（理论上接近 0 最优）

### 3.2 分析要点
- 周转率需结合**信用政策**判断：收紧信用→周转率↑但销售↓
- 存货周转率突然下降→可能积压或产品滞销
- 应付账款周转天数过长→可能损害供应商关系

## 四、盈利能力分析

| 指标 | 公式 | 含义 |
|------|------|------|
| 毛利率 | 毛利润/营业收入 | 产品溢利能力 |
| 净利率 | 净利润/营业收入 | 最终盈利水平 |
| 总资产报酬率(ROA) | 净利润/平均总资产 | 全部资产的盈利能力 |
| 净资产收益率(ROE) | 净利润/平均净资产 | 股东权益回报率 |
| 投入资本回报率(ROIC) | NOPAT/投入资本 | 投资项目的回报率 |
| 每股收益(EPS) | 净利润/普通股股数 | 每股盈利 |
| 市盈率(P/E) | 股价/EPS | 估值水平 |

### 4.1 杜邦分析体系（ROE 三因素分解）
```
ROE = 净利率 × 总资产周转率 × 权益乘数

    = (净利润/营业收入) × (营业收入/总资产) × (总资产/净资产)
```

| 驱动因素 | 含义 | 改进方向 |
|---------|------|---------|
| 净利率 ↑ | 产品盈利能力强或费用控制好 | 提价/降本/优化费用 |
| 周转率 ↑ | 资产使用效率高 | 加快回款/去库存/盘活资产 |
| 权益乘数 ↑ | 财务杠杆高 | 适度负债/谨慎使用 |

### 4.2 分析要点
- ROE > 15% 通常被认为是**优秀**水平（需考虑行业特性）
- ROE 过高（> 40%）需检查是否过度使用财务杠杆
- ROA vs ROE 对比可判断杠杆贡献程度

## 五、成长能力分析

| 指标 | 公式 | 含义 |
|------|------|------|
| 营业收入增长率 | (本期收入-上期收入)/上期收入 | 规模增长 |
| 净利润增长率 | (本期净利-上期净利)/上期净利 | 盈利增长 |
| 总资产增长率 | (期末总资产-期初总资产)/期初总资产 | 资产扩张 |
| 净资产增长率 | (期末净资产-期初净资产)/期初净资产 | 股东权益增长 |
| 可持续增长率 | ROE × (1-股利支付率) | 不增发新股下的最大增长率 |

### 5.1 增长质量判断
- **有质量的增长**：收入增长 + 利润同步增长 + 经营现金流为正
- **低质量增长**：收入增长但毛利率下降 + 应收账款猛增
- **不可持续增长**：收入增速远超可持续增长率 → 需外部融资支撑

## 六、综合评分与输出

### 6.1 雷达图评分法
从四大类中各选取关键指标，与行业均值对比评分：

| 维度 | 指标 | 本公司 | 行业均值 | 评分(1-5) |
|------|------|--------|---------|-----------|
| 偿债 | 流动比率 | 1.8 | 2.0 | 4 |
| 营运 | 应收账款周转天数 | 45 | 60 | 4 |
| 盈利 | ROE | 18% | 12% | 4 |
| 成长 | 收入增长率 | 25% | 10% | 5 |

### 6.2 输出报告格式

```

| 速动比率 |       |         |      |
| 资产负债率 |      |         |      |
| 利息保障倍数 |     |         |      |

## 二、营运能力分析
[同上表格格式]
```

## 七、使用规则

- 所有比率**必须在同一会计准则下计算**，跨准则需说明差异
- 必须标注**数据来源期间**（如"基于2024年年报数据"）
- 评价必须**结合行业特性**，不可用通用标准一刀切
- 单一比率异常不可直接下结论 → 需**多比率交叉验证**
- 杜邦分析应说明各因素变动对 ROE 的**敏感度/贡献度**
- 建议同时使用**同比变化**和**行业对标**两个维度
