# 折旧计算技能

## 技能描述

折旧计算技能用于自动化完成固定资产折旧的计提、调整与账务处理。支持直线法、工作量法、双倍余额递减法、年数总和法等多种折旧方法，可分别按固定资产类别、使用部门、资产卡片等维度计算折旧额。系统可根据资产的原值、预计净残值、使用年限和折旧方法自动生成月度折旧计提凭证，并支持折旧年限变更、减值计提等特殊业务场景。

## 适用场景

- **月度折旧计提**：每月末自动计算所有在用固定资产的应计折旧额，生成批量凭证
- **新增固定资产**：新购入资产入账后，自动计算从入账次月起应计提的折旧
- **资产处置与报废**：资产出售、报废或毁损时，计算截至处置日的累计折旧及账面净值
- **折旧方法变更**：企业会计政策变更时，对既有资产的折旧方法进行变更并计算调整额

## 输入格式

```json
{
  "asset_id": "FIX-2025-001",
  "asset_name": "服务器设备",
  "category": "电子设备",
  "department": "信息技术部",
  "original_value": 120000.00,
  "residual_value": 6000.00,
  "useful_life_years": 5,
  "depreciation_method": "双倍余额递减法",
  "start_date": "2025-06-01",
  "current_period": "2026-05"
}
```

## 输出示例

```
固定资产折旧计算表
期间：2026年5月

资产编号：FIX-2025-001
资产名称：服务器设备
原值：120,000.00元
预计净残值：6,000.00元
使用年限：5年（60个月）
折旧方法：双倍余额递减法

计算过程：
  年折旧率 = 2/5 × 100% = 40%
  已使用月数：11个月（2025.06-2026.04）
  累计已提折旧：120,000 × 40% + (120,000-48,000)×40%/12×11
              = 48,000 + 26,400 = 74,400.00元
  本月应提折旧：(120,000-74,400)×40%/12 = 1,520.00元
  本月末累计折旧：74,400 + 1,520 = 75,920.00元
  账面净值：120,000 - 75,920 = 44,080.00元

生成凭证：
  借：管理费用-折旧费    1,520.00
  贷：累计折旧          1,520.00
```

## 注意事项

- 固定资产当月增加，从次月开始计提折旧；当月减少，当月照提折旧
- 已提足折旧但仍继续使用的资产不再计提折旧，需在资产卡片中标记
- 折旧方法一经确定不得随意变更，如需变更需经董事会批准并在报表附注中披露
- 计提减值准备的资产，其折旧基数应减去已计提的减值准备
- 税法折旧年限与会计折旧年限不同时，需在汇算清缴时做纳税调整
