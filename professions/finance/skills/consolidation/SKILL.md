# 合并报表

## 概述

本技能涵盖企业集团合并财务报表的核心操作，包括合并抵銷分录的编制（长期股权投资抵销、内部交易抵销、内部债权债务抵销）、合并工作底稿的编制方法，以及少数股东权益的计算与列报。合并报表反映的是以企业集团为单一会计主体的财务状况与经营成果。

## 合并范围判断

根据《企业会计准则第33号——合并财务报表》，合并范围以**控制**为基础确定：

- **控制三要素**：主导被投资方的权力、享有可变回报、有能力运用权力影响回报
- **持股比例**：直接或间接持有 **50%以上** 表决权通常视为控制（有反证除外）
- **结构化主体**：即使不持有股权，实际控制权也需纳入合并

## 合并工作底稿

### 工作底稿结构

```
┌────────────┬──────┬──────┬──────┬──────┬──────────────┐
│  报表项目   │ 母公司 │ 子公司 │ 合计  │ 抵销分录 │  合并数      │
│            │      │      │      │ 借 │ 贷 │            │
├────────────┼──────┼──────┼──────┼────┼────┼──────────────┤
│ 营业收入   │ 1000 │  500 │ 1500 │  200│    │      1300    │
│ 减：营业成本│  600 │  300 │  900 │    │ 150 │       750    │
│ 营业利润   │  400 │  200 │  600 │    │    │       550    │
│ ...        │      │      │      │    │    │              │
│ 资产总计   │ 5000 │ 2000 │ 7000 │  50 │ 600│      6450    │
│ 负债合计   │ 2000 │  800 │ 2800 │     │  30│      2770    │
│ 所有者权益 │ 3000 │ 1200 │ 4200 │ 1200│ 200│      3200    │
│ 少数股东权益│      │      │      │     │    │       480    │
└────────────┴──────┴──────┴──────┴──────┴────┴──────────────┘
```

### 编制步骤

1. **个别报表汇总**：将母公司及各子公司报表各项目简单相加（"合计"栏）
2. **编制抵销分录**：按上述抵销原则编制抵销分录
3. **过入工作底稿**：抵销分录分别计入"抵销分录"的借/贷栏
4. **计算合并数**：合计 + 借方抵销 - 贷方抵销（资产、成本费用类）
                    合计 - 借方抵销 + 贷方抵销（负债、权益、收入类）
5. **生成合并报表**：根据工作底稿的"合并数"栏编制正式报表

### 合并数计算规则

| 项目类型 | 计算公式 |
|---------|---------|
| 资产类 | 合计 + 借方抵销 - 贷方抵销 |
| 负债类 | 合计 - 借方抵销 + 贷方抵销 |
| 所有者权益类 | 合计 - 借方抵销 + 贷方抵销 |
| 收入类 | 合计 - 借方抵销 + 贷方抵销 |
| 费用类 | 合计 + 借方抵销 - 贷方抵销 |
| 利润分配类 | 合计 ± 分析确定 |

## 少数股东权益与损益

### 少数股东权益

计算公式：
```
少数股东权益 = 子公司净资产 × 少数股东持股比例
```

列报位置：资产负债表所有者权益项下，与"归属于母公司所有者权益"并列。

### 少数股东损益

计算公式：
```
少数股东损益 = 子公司净利润 × 少数股东持股比例
```

列报位置：合并利润表中，净利润项下列示"少数股东损益"与"归属于母公司所有者的净利润"。

### 超额亏损处理

子公司亏损超过少数股东权益份额时：
- 少数股东权益减记至零为限
- 超额亏损部分由母公司承担
- 恢复盈利时优先恢复少数股东权益

## 合并现金流量表

需抵销的项目：
- 母子公司之间的现金往来（投资/借款/还款）
- 内部购销产生的现金收付
- 子公司支付给母公司的股利

## 常见审计问题

1. **合并范围遗漏**：未纳入实际控制的子公司
2. **内部交易抵销不完整**：往来款项、收入成本未全额抵销
3. **顺/逆流交易处理错误**：逆流交易需同时调整少数股东权益
4. **子公司会计政策不一致**：未按母公司会计政策统一调整
5. **持续计算的商誉减值**：商誉每年需做减值测试

## 最佳实践

1. **统一会计政策**：合并前确保母子公司采用一致的会计政策和会计期间
2. **合并对账机制**：编报前设置内部交易对账环节，确保往来一致
3. **模板化抵销**：将常规抵销分录设置为模板，减少重复工作
4. **分步编制**：先编制合并资产负债表与利润表，再编制现金流量表
5. **底稿留痕**：每笔抵销分录注明依据文件编号，便于审计追溯
6. **信息系统支持**：使用合并报表系统（如久其、元年、Oracle HFM）自动采集数据
