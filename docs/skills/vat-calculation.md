# 增值税计算技能

## 技能描述

增值税计算技能用于自动化处理企业增值税的税额计算、申报表生成与进项抵扣管理。该技能支持一般纳税人和小规模纳税人两种计税模式，涵盖应纳税额计算、进项税额认证、销项税额汇总、减免税政策适用等核心功能。系统可根据企业开票数据和收票数据自动生成增值税纳税申报表主表及附表，并与税务局接口对接进行一键申报。

## 适用场景

- **月度/季度增值税申报**：企业在法定申报期内，自动汇总当期销项与进项数据，生成申报表
- **进项税额认证管理**：扫描发票抵扣联二维码，自动判断是否可抵扣及抵扣比例
- **简易计税与一般计税并行**：兼营不同税率业务的企业，分别计算各业务线的应纳增值税
- **出口退税申报**：生产企业或外贸企业的出口货物免抵退税额计算

## 输入格式

```json
{
  "taxpayer_type": "一般纳税人",
  "period": "2026-04",
  "sales_data": [
    {"tax_rate": "13%", "taxable_sales": 500000.00, "tax_amount": 65000.00},
    {"tax_rate": "6%", "taxable_sales": 200000.00, "tax_amount": 12000.00}
  ],
  "purchase_data": [
    {"tax_rate": "13%", "invoice_amount": 300000.00, "deductible_tax": 39000.00},
    {"tax_rate": "9%", "invoice_amount": 100000.00, "deductible_tax": 9000.00}
  ]
}
```

## 输出示例

```
┌────────────────────────────────────────────┐
│         增值税纳税申报表（一般纳税人）          │
│                   2026年4月                   │
├────────────────┬───────────┬────────────────┤
│ 项目            │ 金额       │ 说明            │
├────────────────┼───────────┼────────────────┤
│ 销项税额        │ 77,000.00 │ 13%+6%应税合计   │
│ 进项税额        │ 48,000.00 │ 已认证抵扣       │
│ 进项税额转出     │ 2,000.00  │ 集体福利转出     │
│ 应纳增值税      │ 27,000.00 │ =77000-48000+2000│
│ 减免税额        │ 0.00      │                  │
│ 本期应补(退)税额 │ 27,000.00 │                  │
└────────────────┴───────────┴────────────────┘
```

## 注意事项

- 不同税率的业务必须分开核算，未分别核算的从高适用税率
- 进项税额认证必须在申报期内完成，逾期不可补认证
- 简易计税项目对应的进项税不得抵扣，需单独核算
- 关注增值税税收优惠政策（如小微企业减免、加计抵减等）的适用条件
- 发票信息与申报数据的一致性校验至关重要，建议每月自动比对开票系统与财务账套数据
