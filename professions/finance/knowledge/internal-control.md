# 企业内部控制基础

> 内部控制不是束缚效率的枷锁，是防止出事的保险。面向中小企业，重点讲实用的控制措施。

---

## 一、内部控制五大要素（COSO 框架）

| 要素 | 核心问题 | 中小企业怎么做 |
|------|---------|--------------|
| **控制环境** | 公司上下的基调——老板自己是否重视合规？ | 老板签字审批时问一句"这合规吗？"比100条制度有用 |
| **风险评估** | 你最怕出什么事？ | 列一个"风险清单"，按可能性和影响程度排序 |
| **控制活动** | 有哪些制度防止出错/舞弊？ | 不兼容岗位分离、审批流程、定期盘点 |
| **信息与沟通** | 信息能顺利传到该知道的人那里吗？ | 周会、微信群同步、异常即时上报 |
| **监控** | 这些控制措施有没有在运转？ | 每月检查一次关键控制点的执行 |

## 二、核心控制措施

### 2.1 不兼容职务分离（最重要的一条）

**原则**：一个人不能从头到尾包办一项经济业务。

| 禁止一人兼任的组合 | 为什么 |
|-------------------|--------|
| **收款 + 记账** | 极易隐匿收入、挪用资金 |
| **采购下单 + 收货验收** | 可能买次充好、吃回扣 |
| **出纳 + 会计** | 管钱的不管账，管账的不管钱 |
| **发工资 + 算工资** | 可能给不存在的人发工资（"吃空饷"） |
| **签订合同 + 审批合同** | 自己和自己签合同 |

> 如果公司小、人手不够怎么办 → 至少做到：**老板亲自审批付款 + 定期抽查**

### 2.2 授权审批

| 金额等级 | 审批人 |
|---------|--------|
| 日常小额（< 500元） | 部门负责人 |
| 中等金额（500-5000元） | 分管副总 |
| 大额（5000-50000元） | 总经理/CEO |
| 重大（> 50000元） | 董事会/老板 |

**审批前必须看的三样东西**：
1. 为什么花这笔钱？（必要性）
2. 有没有比价？（至少2家）
3. 是否有预算？（超预算走追加流程）

### 2.3 资产保护

| 资产类型 | 控制措施 |
|---------|---------|
| **现金** | 日清月结、保险柜双人共管、每天存银行不要留大额现金 |
| **银行存款** | 网银U盾分人保管（制单+审核分开）、月度银行余额调节表 |
| **存货** | 定期盘点（至少每季度）、出入库要有签字的单据 |
| **固定资产** | 建立台账、贴标签、每年全面盘点 |
| **应收账款** | 信用额度管理、账龄分析、逾期催收制度 |
| **公章/合同章** | 用印登记、双人管理、禁止带出公司 |

### 2.4 采购与付款控制

```
请购 → 比价/招标 → 采购 → 验收 → 付款
 ↑ 需审批    ↑ 至少2家  ↑ 下单  ↑ 验数量质量 ↑ 见发票+入库单才付
```

**采购常见舞弊信号**：
- 长期只从一家供应商采购，价格明显高于市场价
- 采购金额刚卡在审批线以下（规避审批）
- 同一家供应商频繁小额采购

## 三、销售与收款控制

```
客户信用评估 → 签订合同 → 发货 → 开票 → 收款 → 对账
                                ↑ 见发货单才开票    ↑ 定期与客户对账
```

**关键控制点**：
- 赊销前必须评估客户信用（大客户 ≠ 不拖欠）
- 信用期到期未回款 → 停止发货
- 每周更新应收账款账龄表
- 超过账期 90 天的重点催收

## 四、财务报告控制

### 4.1 月末结账检查清单

| 检查项 | 标准 |
|--------|------|
| 银行对账 | 银行余额调节表差异为0 |
| 应收账款 | 与销售部门的发货记录核对 |
| 应付账款 | 与采购入库记录核对 |
| 存货 | 账面与盘点一致 |
| 固定资产 | 折旧计提正确、新增/报废处理 |
| 收入成本配比 | 当月收入对应的成本是否全部结转 |
| 关联交易 | 是否按独立交易原则处理 |
| 异常科目 | "其他应收款""其他应付款"中是否有不合理挂账 |

### 4.2 财务舞弊信号

| 信号 | 可能的问题 |
|------|-----------|
| 收入和利润高速增长但经营现金流为负 | 应收账款造假、虚构收入 |
| 毛利率远超行业水平 | 虚增收入或漏记成本 |
| 存货异常增长但周转率下降 | 积压或成本未结转 |
| "其他应收款"挂账金额大且长期不动 | 大股东挪用资金 |
| 频繁更换审计师 | 审计意见不理想 |
