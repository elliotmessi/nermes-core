# 税务申报技能

## 技能描述

税务申报技能用于自动化完成企业各项税种的纳税申报流程，包括数据采集、申报表生成、税务逻辑校验及申报提交。技能覆盖增值税、企业所得税、个人所得税、印花税、房产税、土地使用税、城市维护建设税及附加等主要税种。系统支持与金税系统、电子税务局的数据对接，实现从财务数据到申报数据的自动转换和一键申报。

## 适用场景

- **按月/按季常规申报**：增值税、附加税、个税等固定周期的自动申报
- **企业所得税汇算清缴**：年度企业所得税的纳税调整、减免税计算与申报
- **新设企业税务登记**：企业在成立初期完成税种核定、发票申领等税务初始化工作
- **税务稽查配合**：为税务稽查准备完整的申报历史、纳税调整说明和相关凭证资料

## 注意事项

- 企业所得税申报需与会计报表利润进行比对，确认税会差异调整的合理性
- 增值税发票勾选认证需在申报前完成，建议设置截止日自动提醒
- 享受税收优惠必须留存备查资料（如高新技术企业证书、研发费用辅助账等）
- 延期申报需提前申请，逾期申报将产生滞纳金并影响纳税信用评级
- 各税种申报截止日不同，建议使用日历提醒功能避免遗漏
