# 财务报表分析核心指标速查

> 面向企业财务人员的常用财务比率与分析框架。所有指标附计算公式和判断基准。

---

## 一、盈利能力指标

### 1.1 毛利率

```
毛利率 = (营业收入 - 营业成本) / 营业收入 × 100%
```

| 判断 | 制造业 | 零售业 | 软件/SaaS |
|------|--------|--------|-----------|
| 优秀 | >40% | >35% | >70% |
| 良好 | 25-40% | 20-35% | 50-70% |
| 一般 | 15-25% | 10-20% | 30-50% |
| 差 | <15% | <10% | <30% |

> 毛利率下降通常意味着：原材料涨价未传导、竞争加剧被迫降价、产品结构恶化。

### 1.2 净利率

```
净利率 = 净利润 / 营业收入 × 100%
```

| 行业 | 参考值 |
|------|--------|
| 制造业 | 5-15% |
| 零售/批发 | 1-5% |
| 互联网/SaaS | 15-30% |
| 金融 | 20-40% |

> 净利率远低于毛利率 → 期间费用过高，需要排查管理费用和销售费用的合理性。

### 1.3 ROE（净资产收益率）

```
ROE = 净利润 / 平均净资产 × 100%
```

杜邦分解：
```
ROE = 净利率 × 总资产周转率 × 权益乘数
    = (净利润/收入) × (收入/总资产) × (总资产/净资产)
```

- ROE > 15%：优秀
- ROE 10-15%：良好
- ROE < 5%：需关注

> 高 ROE 可能来自高杠杆（权益乘数大）——要区分是经营能力还是财务风险。

### 1.4 ROA（总资产收益率）

```
ROA = 净利润 / 平均总资产 × 100%
```

- ROA > 10%：资产利用效率高
- ROA 5-10%：正常
- ROA < 3%：资产回报偏低

## 二、偿债能力指标

### 2.1 流动比率

```
流动比率 = 流动资产 / 流动负债
```

- > 2.0：流动性充足
- 1.5-2.0：正常
- 1.0-1.5：偏紧，需关注
- < 1.0：短期偿债风险（但零售/餐饮等行业可 < 1.0）

### 2.2 速动比率

```
速动比率 = (流动资产 - 存货) / 流动负债
```

- > 1.0：安全
- 0.6-1.0：正常
- < 0.5：短期偿债压力

> 存货占比高的企业（制造业、零售），速动比率比流动比率更有参考意义。

### 2.3 资产负债率

```
资产负债率 = 总负债 / 总资产 × 100%
```

| 行业 | 合理范围 |
|------|---------|
| 制造业 | 40-60% |
| 房地产 | 60-80% |
| 互联网/轻资产 | 20-40% |
| 金融 | 80-90%+ |

> 超过 70% 要关注。但不是越低越好——适度负债可利用财务杠杆提升 ROE。

### 2.4 利息保障倍数

```
利息保障倍数 = (利润总额 + 利息费用) / 利息费用
```

- > 5：安全
- 2-5：正常
- < 2：偿息压力大，银行可能要求追加担保

## 三、营运能力指标

### 3.1 应收账款周转率

```
应收账款周转率 = 营业收入 / 平均应收账款
应收账款周转天数 = 365 / 周转率
```

| 行业 | 参考周转天数 |
|------|------------|
| 制造业 | 60-90天 |
| 零售 | 3-15天 |
| 工程/建筑 | 120-180天 |
| SaaS（预收款模式） | 负值或极低 |

> 周转天数突然拉长 → 可能客户回款困难、或为冲业绩放宽了信用政策。

### 3.2 存货周转率

```
存货周转率 = 营业成本 / 平均存货
存货周转天数 = 365 / 周转率
```

| 行业 | 参考周转天数 |
|------|------------|
| 生鲜/快消 | 3-15天 |
| 一般制造业 | 30-90天 |
| 高端制造/重工 | 90-180天 |
| 白酒 | 200-500天（基酒储存期长） |

> 存货周转天数增加 → 可能积压滞销、生产计划失误、或原材料囤货。

### 3.3 总资产周转率

```
总资产周转率 = 营业收入 / 平均总资产
```

- > 1.0：资产运营效率高（轻资产企业常见）
- 0.5-1.0：正常
- < 0.5：重资产企业常态，但需关注资产回报

## 四、现金流量分析

### 4.1 三大现金流健康组合

| 经营现金流 | 投资现金流 | 筹资现金流 | 阶段判断 |
|-----------|-----------|-----------|---------|
| + | - | + | **成长期**：经营盈利 + 扩张投资 + 外部融资 |
| + | - | - | **成熟期**：经营盈利 + 扩张投资 + 偿还债务/分红 |
| + | + | - | **成熟后期**：经营盈利 + 处置资产 + 大力还债 |
| - | + | + | **转型/困境**：经营亏损 + 变卖资产 + 靠融资续命 |
| - | - | + | **初创/危机**：经营亏损 + 仍在扩张 + 严重依赖融资 |

### 4.2 现金流量比率

```
现金流量比率 = 经营活动现金净流量 / 流动负债
```

- > 0.5：经营现金流可覆盖一半流动负债
- < 0.2：现金流紧张，需关注

### 4.3 盈利现金比率

```
盈利现金比率 = 经营活动现金净流量 / 净利润
```

- > 1.0：利润含金量高，收款好
- 0.5-1.0：正常
- < 0.3：利润可能只是"纸面利润"——大量赊销未回款

## 五、综合分析框架

### 5.1 快速诊断清单

| 异常信号 | 可能问题 |
|---------|---------|
| 毛利率↓ + 存货周转天数↑ | 产品滞销，被迫降价 |
| 净利率正常 + 经营现金流为负 | 赊销严重，回款差 |
| ROE高 + 资产负债率 > 80% | 高杠杆驱动，财务风险大 |
| 收入增长 + 应收账款增速远超收入增速 | 通过放宽信用冲业绩（可能应收账款坏账） |
| 净利润为正 + 经营现金流持续为负 | 利润质量差 |

### 5.2 同行业对比

分析单一指标时要对标同行业、同体量的企业。不同行业的"正常值"差异巨大，不能机械套用。

> 本文为财务分析参考框架。实际分析需结合行业特点、企业生命周期、宏观环境综合判断。
