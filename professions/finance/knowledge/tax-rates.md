# 2026年中国税法速查手册

> 面向财务从业者的核心税种税率与优惠政策速查，适用于一般企业日常税务处理。

---

## 一、增值税（Value-Added Tax, VAT）

增值税是中国第一大税种，对商品和服务流转环节的增值额征税。

### 1.1 税率体系

| 纳税人类型 | 税率/征收率 | 适用范围 |
|-----------|------------|---------|
| **一般纳税人** | **13%** | 销售货物、加工修理修配劳务、有形动产租赁服务 |
| | **9%** | 不动产（房屋、土地）销售/租赁、建筑/运输/邮政/基础电信服务、农产品/图书/天然气等 |
| | **6%** | 现代服务业（研发、信息技术、鉴证咨询、文化创意）、金融、生活服务、增值电信服务 |
| | **0%** | 出口货物、国际运输服务、航天运输服务 |
| **小规模纳税人** | **1%**（2026年延续） | 原3%征收率，2026年继续按1%减征（政策截至2026年12月31日） |
| | **5%** | 不动产销售/租赁（小规模）、劳务派遣差额纳税 |

> **小规模纳税人标准**：年应征增值税销售额 ≤ 500万元（连续12个月或4个季度）。
> **2026年小规模优惠**：月销售额 ≤ 10万元（季度 ≤ 30万元）免征增值税。

### 1.2 进项税额抵扣要点

- **可抵扣**：取得增值税专用发票、海关进口增值税专用缴款书、农产品收购发票（计算抵扣）、通行费电子发票等。
- **不可抵扣**：简易计税项目、集体福利（食堂、班车）、个人消费、非正常损失、贷款服务利息（贷款服务进项税不得抵扣）。
- **认证期限**：自开具之日起360日内认证或登录确认（2022年起已取消认证期限限制，但建议及时处理）。

### 1.3 增值税附加税费

附加税以实际缴纳的增值税为计税依据：

| 税种 | 税率 | 说明 |
|-----|------|------|
| **城市维护建设税** | 7% / 5% / 1% | 市区7%，县城/镇5%，其他地区1% |
| **教育费附加** | 3% | 统一费率 |
| **地方教育附加** | 2% | 统一费率（部分地区可减半至1%） |

> **小型微利企业附加税减半**：2022年1月1日—2026年12月31日，按月纳税的月销售额 ≤ 10万元的小规模纳税人免征教育费附加和地方教育附加；所有小微企业的附加税可减半征收。

### 1.4 增值税申报期限

- **一般纳税人**：按月申报，次月15日前完成。
- **小规模纳税人**：按季度申报，季度结束后次月15日前完成。
- 申报渠道：电子税务局（网页版或客户端）。

## 二、企业所得税（Corporate Income Tax, CIT）

### 2.1 基本税率

| 企业类型 | 税率 | 条件 |
|---------|------|------|
| **一般企业** | **25%** | 标准税率 |
| **高新技术企业** | **15%** | 国家重点支持的高新技术领域，经认定后享受 |
| **小型微利企业** | **5%（实际）** | 年应纳税所得额 ≤ 300万元、从业人数 ≤ 300人、资产总额 ≤ 5000万元 |
| **西部大开发鼓励类企业** | **15%** | 设在西部12省（区、市），主营业务占收入总额60%以上 |
| **海南自贸港鼓励类企业** | **15%** | 注册在海南自贸港并实质性运营 |
| **集成电路/软件企业** | **"两免三减半"等** | 特定优惠期政策，需认定 |

### 2.2 小型微利企业实际税负计算（2026年）

| 年应纳税所得额区间 | 计算方式 | 实际综合税率 |
|------------------|---------|------------|
| ≤ 100万元部分 | 减按25%计入应纳税所得额，按20%税率缴纳 | **5%**（25%×20%） |
| 100万—300万元部分 | 减按25%计入应纳税所得额，按20%税率缴纳 | **5%**（25%×20%） |

> 示例：某小微企业年利润250万元，应纳所得税 = 250万 × 25% × 20% = **12.5万元**（非25%税率下的62.5万元）。

### 2.3 研发费用加计扣除

| 企业类型 | 加计扣除比例 | 说明 |
|---------|-------------|------|
| **一般企业** | 100% | 实际发生研发费用的100%额外税前扣除 |
| **集成电路/工业母机企业** | 120% | 2023—2027年，额外扣除120% |

> 例如：发生研发支出100万元，一般企业税前可扣除200万元（100万实际支出 + 100万加计），节税效果显著。2026年制造业与非制造业企业统一适用100%。

### 2.4 其他重要优惠

- **设备器具一次性扣除**：新购单位价值 ≤ 500万元的设备、器具，可在税前一次性扣除。
- **残疾人工资加计扣除**：支付给残疾人的实际工资，可100%加计扣除。
- **环境保护、节能节水、安全生产专用设备**：投资额的10%可从应纳税额中抵免，不足部分可结转5年。
- **创业投资企业**：投资于未上市中小高新技术企业满2年的，可按投资额的70%抵扣应纳税所得额。

### 2.5 企业所得税申报期限

- **预缴**：按月或按季度预缴（一般企业按季度），季度结束后15日内预缴。
- **汇算清缴**：年度终了后5个月内（次年1月1日—5月31日），向税务机关报送年度申报表并结清税款。

## 三、个人所得税（Individual Income Tax, IIT）

### 3.1 综合所得（工资薪金、劳务报酬、稿酬、特许权使用费）

采用**七级超额累进税率**，年计税（2026年基本减除费用为60,000元/年，即5,000元/月）：

| 级数 | 全年应纳税所得额 | 税率 | 速算扣除数 |
|-----|----------------|------|-----------|
| 1 | 不超过36,000元 | 3% | 0 |
| 2 | 36,001—144,000元 | 10% | 2,520元 |
| 3 | 144,001—300,000元 | 20% | 16,920元 |
| 4 | 300,001—420,000元 | 25% | 31,920元 |
| 5 | 420,001—660,000元 | 30% | 52,920元 |
| 6 | 660,001—960,000元 | 35% | 85,920元 |
| 7 | 超过960,000元 | 45% | 181,920元 |

**专项附加扣除标准**（2026年执行）：

| 扣除项目 | 扣除标准 |
|---------|---------|
| 子女教育 | 每个子女每月2,000元（3岁至博士，含学前教育） |
| 继续教育 | 学历教育每月400元；职业资格取得当年3,600元 |
| 大病医疗 | 医保报销后个人自付超15,000元部分，上限80,000元/年 |
| 住房贷款利息 | 首套房每月1,000元，最长240个月 |
| 住房租金 | 800/1,100/1,500元/月（按城市等级） |
| 赡养老人 | 独生子女每月3,000元，非独生分摊 |
| 3岁以下婴幼儿照护 | 每个婴幼儿每月2,000元 |

### 3.2 经营所得（个体工商户、个人独资企业）

- 五级超额累进税率：5%—35%
- 年应纳税所得额不超过30,000元：5%
- 超过30,000—90,000元：10%
- 超过90,000—300,000元：20%
- 超过300,000—500,000元：30%
- 超过500,000元：35%

### 3.3 其他分类所得

| 所得类型 | 税率 |
|---------|------|
| 利息、股息、红利所得 | 20% |
| 财产租赁所得 | 20% |
| 财产转让所得 | 20% |
| 偶然所得（中奖等） | 20% |

## 四、印花税（Stamp Duty）

2022年7月1日新《印花税法》实施后主要变化：

| 应税凭证 | 税率 |
|---------|------|
| 买卖合同（购销合同） | 万分之三（0.3‰） |
| 借款合同 | 万分之零点五（0.05‰） |
| 技术合同 | 万分之三（0.3‰） |
| 财产租赁合同 | 千分之一（1‰） |
| 运输合同 | 万分之三（0.3‰） |
| 仓储保管合同 | 千分之一（1‰） |
| 产权转移书据（股权、不动产） | 万分之五（0.5‰） |
| 营业账簿 | 万分之二点五（0.25‰） |
| **证券交易**（A股/B股） | **千分之一（1‰）**，单边征收（仅卖出方） |

> **小规模纳税人/小微企业**：印花税可减半征收（至2027年12月31日）。

## 五、房产税（Property Tax）

| 计税方式 | 计税依据 | 税率 |
|---------|---------|------|
| 自用房产 | 房产原值一次减除10%—30%后的余值 | 1.2% |
| 出租房产 | 租金收入（不含增值税） | 12% |
| 个人出租住房 | 租金收入 | 4%（减按） |

> 房产税按年计算、分期缴纳（各地规定不同，通常按季度或半年）。

---

## 六、土地增值税（Land Appreciation Tax）

**超率累进税率**，适用于转让国有土地使用权及地上建筑物：

| 增值额占扣除项目比例 | 税率 | 速算扣除系数 |
|--------------------|------|------------|
| ≤ 50%部分 | 30% | 0% |
| 50%—100%部分 | 40% | 5% |
| 100%—200%部分 | 50% | 15% |
| > 200%部分 | 60% | 35% |

> 房地产开发企业可享受20%的加计扣除（开发成本）。

## 七、其他税种速查

| 税种 | 税率/标准 | 计税依据 |
|-----|----------|---------|
| **契税** | 3%—5%（各省浮动，首套房优惠至1%—1.5%） | 不动产成交价格 |
| **城镇土地使用税** | 0.6—30元/平方米/年（按城市等级） | 实际占用土地面积 |
| **车辆购置税** | 10% | 车辆不含税价格 |
| **消费税** | 比例税率+定额税率（烟、酒、化妆品、汽车等15类） | 销售额或销售数量 |
| **资源税** | 1%—20%不等 | 应税资源销售额/销售量 |
| **环境保护税** | 按污染物种类定额 | 大气/水/固废/噪声排放量 |

## 八、申报期限总结

| 税种 | 申报周期 | 截止时间 |
|-----|---------|---------|
| 增值税（一般纳税人） | 月 | 次月15日 |
| 增值税（小规模） | 季 | 季度后次月15日 |
| 企业所得税预缴 | 季 | 季度后15日 |
| 企业所得税汇算清缴 | 年 | 次年5月31日 |
| 个人所得税（工资） | 月（代扣代缴） | 次月15日 |
| 个人所得税汇算清缴 | 年 | 次年6月30日 |
| 印花税 | 季/年（据实或核定） | 季度后15日/次年1月15日 |
| 房产税 | 年（分期） | 各地不同（多为每季度末） |
| 附加税 | 与增值税同步 | 与增值税相同 |

> **延期申报**：因不可抗力可申请延期，最长不超过3个月。税款需预缴。

## 九、区域性优惠政策专题

### 海南自由贸易港

- **鼓励类产业**：企业所得税15%（2025—2027年持续）
- **高端紧缺人才**：个人所得税超过15%部分免征
- **封关运作后**：全岛零关税（"一线放开、二线管住"）

### 西部大开发

- 对西部地区12省（区、市）鼓励类产业企业，减按15%征收企业所得税
- 主营业务收入占企业总收入的60%以上方可享受

### 上海自贸区临港新片区

- 重点产业（集成电路、人工智能、生物医药等）企业所得税15%

## 十、2026年税务工作提醒

1. **数电票全面推广**：2026年全面数字化电子发票已覆盖全国，不再使用纸质专票。
2. **金税四期**：以数治税——发票、社保、银行流水等多维数据交叉比对，合规要求显著提高。
3. **社保入税**：社保费由税务部门统一征收，工资个税、社保、公积金须口径一致。
4. **留抵退税常态化**：符合条件的纳税人可申请增值税留抵退税，不再限于特定行业。
5. **税务合规重点**：虚开发票（特别是数电票虚开）、隐匿收入、公私混用、个独企业/合伙企业核定征收滥用等。
