# 客服话术技能 — 售前转化与售后服务

## 功能描述
提供电商客服全流程话术支持，包括售前咨询、催付、售后处理、差评回复等场景，帮助提升客服转化率、响应速度和客户满意度。

## 核心能力

### 1. 售前话术
- **主动问候**：根据进店来源定制开场白（搜索进店/活动进店/直播进店）
- **需求挖掘**：通过提问了解客户需求（使用场景、预算、偏好）
- **关联推荐**：基于已选商品推荐搭配品、升级款，提升连带率
- **尺码/规格推荐**：根据用户信息推荐最合适的规格
- **价格疑虑处理**：解释性价比、对比竞品、强调附加价值
- **催付话术**：库存紧张提醒、优惠倒计时、赠品即将送完

### 2. 售后话术
- **退换货流程**：清晰说明退换货政策和操作步骤
- **质量问题处理**：诚恳道歉 + 提供方案（补发/退款/补偿）
- **物流问题**：主动查询物流 + 安抚等待情绪 + 提供补偿
- **纠纷预防**：换位思考，避免激化矛盾

### 3. 差评回复模板
- **态度诚恳**：感谢反馈 + 承认不足 + 说明改进
- **解决问题**：邀请联系客服 + 提供具体补偿
- **展示给其他买家看**：体现店铺负责任的态度

### 4. 自动回复配置
- **关键词自动回复**：运费、发货时间、尺码、退换货等高频问题
- **快捷短语**：常用话术预设，一键发送
- **分流规则**：售前/售后分流、VIP客户优先接入

### 5. 客服数据分析
- **响应时长**：首次响应 ≤30秒是优秀水平
- **客服转化率**：咨询后成交比例，行业平均20%-40%
- **满意度评分**：4.8分以上为优秀

## 使用场景
- 大促期间客服话术批量准备
- 差评回复优化
- 新客服培训话术
- 自动回复机器人配置
- 客服数据复盘

## 输入
咨询场景描述、商品信息、客户问题类型、当前话术（可选）

## 输出
针对性的客服话术方案、快捷回复模板、差评回复建议、自动回复配置方案
