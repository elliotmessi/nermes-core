# 增值税计算技能

本技能提供增值税计算全流程指导，涵盖一般纳税人与小规模纳税人的计税方法、进项销项管理以及留抵退税操作。

---

## 1. 纳税人身份判断

### 1.1 一般纳税人
- **认定标准**：年应税销售额超过 500 万元（工业企业 / 商业企业），或会计核算健全且自愿申请。
- **计税方法**：一般计税方法（销项税额 － 进项税额）。
- **适用税率**：13%（货物/劳务）、9%（不动产/农产品等）、6%（服务/无形资产）、0%（出口）。

### 1.2 小规模纳税人
- **认定标准**：年应税销售额 ≤ 500 万元，会计核算不健全。
- **计税方法**：简易计税方法（销售额 × 征收率）。
- **适用征收率**：3%（默认）、5%（不动产租赁/转让等）、减按 1%（阶段性减免政策期间）。

---

## 2. 一般纳税人增值税计算

### 2.1 销项税额计算

```
销项税额 = 销售额 × 适用税率
销售额   = 含税销售额 ÷ (1 + 税率)
```

**操作步骤：**
1. 确认当期应税销售收入（含税/不含税）。
2. 若含税，先换算为不含税销售额。
3. 按适用税率计算销项税额。

### 2.2 进项税额计算

```
可抵扣进项税额 = ∑(各采购项目不含税金额 × 扣税凭证注明的税额)
```

**可抵扣凭证类型：**
- 增值税专用发票（抵扣联）
- 海关进口增值税专用缴款书
- 农产品收购发票 / 销售发票（计算抵扣）
- 代扣代缴税收完税凭证
- 通行费电子普通发票（注明税额部分）
- 国内旅客运输服务（电子普通发票/注明旅客身份信息的票据计算抵扣）

**不可抵扣情形：**
- 简易计税方法项目
- 免征增值税项目
- 集体福利或个人消费
- 非正常损失
- 贷款服务（利息支出）

### 2.3 应纳税额计算

```
应纳税额 = 销项税额 － 当期可抵扣进项税额 － 留抵税额
```

## 3. 小规模纳税人增值税计算

```
应纳税额 = 不含税销售额 × 征收率
不含税销售额 = 含税销售额 ÷ (1 + 征收率)
```

**注意事项：**
- 小规模纳税人不得抵扣进项税额。
- 月销售额 ≤ 10 万元（季度 ≤ 30 万元）免征增值税（现行政策，以最新文件为准）。
- 适用 3% 征收率的应税销售收入，减按 1% 征收（阶段性政策）。

## 4. 进项销项管理

### 4.1 进项管理流程
1. **发票采集**：收到增值税专用发票后，及时登录增值税发票综合服务平台（或电子税务局）进行发票认证。
2. **认证勾选**：在平台中查询发票 → 确认发票信息无误 → 勾选认证。
3. **数据归集**：按月份归类汇总进项税额，区分可抵扣与不可抵扣项目。
4. **台账登记**：建立进项税额台账，记录发票号码、金额、税额、认证日期、用途分类。

### 4.2 销项管理流程
1. **收入确认**：按纳税义务发生时间确认应税收入（并非开票时间）。
2. **开票管理**：核对销售合同与发货单据 → 开具增值税发票（专票/普票/电子票）。
3. **税率核对**：确保适用税率正确，区分混合销售与兼营。
4. **台账登记**：建立销项税额台账，按发票号码归档。

### 4.3 常见风险点
- 进项发票未在 360 日内认证（已取消认证期限，但建议及时处理）。
- 发票内容与合同/资金流不一致（三流一致风险）。
- 进项项目用于集体福利/个人消费却已抵扣。
- 混合销售中低税率适用错误。

## 5. 留抵退税处理

### 5.1 适用条件
- 纳税人信用等级为 A 级或 B 级。
- 申请退税前 36 个月未发生骗取留抵退税、出口退税或虚开增值税专用发票情形。
- 申请退税前 36 个月未因偷税被税务机关处罚两次及以上。
- 自 2019 年 4 月 1 日起未享受即征即退、先征后返（退）政策。

### 5.2 计算方式

```
增量留抵退税额 = 增量留抵税额 × 进项构成比例 × 60%  （一般企业）
全额退还         = 增量留抵税额 × 进项构成比例           （先进制造业/符合条件的行业）
```

**进项构成比例 =** 2019年4月至申请退税前一税款所属期内，已抵扣的增值税专用发票（含税控机动车销售统一发票）、海关进口增值税专用缴款书、解缴税款完税凭证注明的增值税额之和 ÷ 同期全部已抵扣进项税额之和。

### 5.3 申请流程
1. 确认当期期末留抵税额。
2. 计算可申请退税额。
3. 登录电子税务局 → 进入"增值税留抵退税"模块。
4. 填写《退（抵）税申请表》。
5. 提交申请并跟踪审批进度。
6. 收到退税款项后进行账务处理（冲减"应交税费——应交增值税（留抵税额）"）。

### 5.4 账务处理示例
```
// 收到退税款时
借：银行存款
  贷：应交税费——应交增值税（进项税额转出）/ 应交税费——增值税留抵税额
```

## 6. 综合计算示例

### 示例：一般纳税人
某企业 2025 年 5 月数据：
- 不含税销售额 500 万元，适用税率 13%
- 取得增值税专用发票注明进项税额 40 万元
- 上月留抵税额 5 万元

```
销项税额 = 500 × 13% = 65 万元
应纳增值税 = 65 - 40 - 5 = 20 万元
```

### 示例：小规模纳税人
某小规模企业 2025 年 4 月数据：
- 含税销售额 30.9 万元，征收率 3%（减按 1%）

```
不含税销售额 = 30.9 ÷ (1 + 1%) = 30.594 万元
应纳税额 = 30.594 × 1% = 0.306 万元
```

## 7. 税务合规要点

1. **纳税申报期限**：增值税按月/按季申报，一般纳税人为月报，小规模可选月报或季报。申报期限为次月 15 日内。
2. **发票管理**：使用增值税发票管理系统（税控盘/UKey）或电子税务局开具发票。
3. **账簿保留**：增值税相关账簿、凭证至少保留 10 年。
4. **关联交易**：关联交易定价需符合独立交易原则，避免被税务机关特别纳税调整。
