# 凭证录入技能

## 技能描述

凭证录入技能用于自动完成会计凭证的创建、审核与归档。该技能能够解析各类原始单据（如发票、收据、银行回单等），按照会计准则自动生成借贷分录，并支持批量导入与模板化录入。系统内置智能校验引擎，可在录入过程中实时检测科目匹配、金额平衡、辅助核算等关键要素的合规性，大幅减少人工录入错误。

## 适用场景

- **日常账务处理**：企业财务部门每日产生的大量费用报销、采购付款、销售收入等凭证的标准化录入
- **批量数据迁移**：从旧财务系统切换到新系统时，历史凭证数据的批量导入与格式转换
- **移动端快速报账**：员工通过手机拍照上传发票，系统自动识别并生成凭证草稿
- **月末结转处理**：期末结转损益、计提折旧、摊销待摊费用等周期性凭证的自动化处理

## 输入格式

支持以下输入方式：

1. **结构化 JSON 输入**：
```json
{
  "voucher_date": "2026-05-23",
  "attachment_count": 2,
  "summary": "采购办公用品",
  "entries": [
    {"account_code": "6602", "direction": "debit", "amount": 5000.00, "summary": "管理费用-办公费"},
    {"account_code": "1002", "direction": "credit", "amount": 5000.00, "summary": "银行存款"}
  ]
}
```

2. **自然语言描述**：如"报销张三差旅费3000元，包括机票2000元和住宿1000元，从公司账户支付"

3. **图片/PDF 附件上传**：支持 OCR 识别发票、收据、合同等原始凭证图像

## 输出示例

```json
{
  "voucher_id": "记-2026-05-23-001",
  "status": "已审核",
  "entries": [
    {"account_name": "管理费用-差旅费", "debit": 3000.00, "credit": 0.00},
    {"account_name": "银行存款", "debit": 0.00, "credit": 3000.00}
  ],
  "balance_check": "通过",
  "audit_trail": "2026-05-23 09:30:00 系统自动生成 | 2026-05-23 09:30:01 智能校验通过"
}
```

## 注意事项

- 凭证录入后不可直接删除，应采用红字冲销或补充登记方式更正
- 涉及现金、银行存款科目的凭证需确保当日余额充足
- 外币业务需同时录入原币金额和本位币金额，系统自动计算汇兑损益
- 建议开启"借贷平衡实时校验"功能，在保存前拦截不平衡凭证
- 辅助核算项目（部门、项目、客户、供应商等）应尽可能完整填写，便于后续多维分析
