# 企业现金流管理

> 现金流是企业的血液。本指南涵盖现金流预测、资金规划、融资策略。

---

## 一、现金流的三条生命线

### 1.1 资金存量的三个层次

| 层次 | 含义 | 安全标准 |
|------|------|---------|
| **现金存量** | 银行存款+库存现金 | 覆盖3-6个月固定支出 |
| **速动资金** | 现金+应收票据+短期理财 | 覆盖6-12个月固定支出 |
| **可用授信** | 银行已批未用的贷款额度 | 随时可动用的"后备金" |

> 很多企业账上利润可观却突然倒闭——不是因为亏损，是因为现金流断裂。

### 1.2 现金流预警信号

| 信号 | 严重程度 | 应对 |
|------|---------|------|
| 连续两个月经营现金流为负 | ⚠️ 黄色 | 排查原因，是季节性还是系统性 |
| 靠短期借款发工资 | 🔴 红色 | 立即缩减开支+催收应收款+寻找融资 |
| 无法按时支付供应商款 | 🔴 红色 | 与供应商协商延期+加快库存周转 |
| 银行贷款到期无法周转 | 🆘 危急 | 找过桥资金+申请展期+变卖资产 |

## 二、现金流预测

### 2.1 短期预测（13周滚动的现金流表）

| 周 | 现金流入 | 现金流出 | 净流量 | 期末余额 |
|----|---------|---------|--------|---------|
| 第1周 | 回款预计+其他 | 工资+税款+采购+费用 | | |
| 第2周 | ... | ... | | |
| ... | | | | |
| 第13周 | | | | |

**流入项**：
- 应收账款预计回款（按账期+历史回款率）
- 预收账款/合同负债到账
- 其他（退税、补贴、投资收益）

**流出项**：
- 工资及社保（固定，最优先保障）
- 税款（有法定截止日，不可拖延）
- 采购付款（按合同账期）
- 房租/水电/运维费
- 利息/手续费

### 2.2 中长期预测（年度）

```
年度现金流入 - 年度现金流出 = 年度现金净流量
 + 期初现金余额
 = 期末现金余额
```

如果期末现金余额为负 → 需要提前安排融资，别等到没钱了才找银行。

## 三、营运资金管理

### 3.1 现金转换周期

```
现金转换周期 = 存货周转天数 + 应收账款周转天数 - 应付账款周转天数
```

| 周期含义 | 越短越好 | 越长越好 |
|---------|---------|---------|
| 存货周转天数 | ✅ 快销快转 | |
| 应收账款周转天数 | ✅ 快速回款 | |
| 应付账款周转天数 | | ✅ 推迟付款 |
| **现金转换周期** | ✅ **越短，资金效率越高** | |

> 举例：存货30天 + 收款60天 - 付款45天 = 现金被占用45天。如果能压缩存货到20天、用供应链金融提前回款、争取供应商60天账期 → 周期降到20天。

### 3.2 各环节优化策略

| 环节 | 优化手段 |
|------|---------|
| **加速回款** | 缩短账期、现金折扣（2/10 n/30）、预收款模式、保理/应收款融资 |
| **延缓付款** | 争取更长账期、用银行承兑汇票代替现金支付 |
| **减少存货占用** | JIT采购、安全库存合理设定、滞销品及时清理 |
| **集中资金管理** | 多账户→集中收付、资金池、日终归集 |

## 四、融资工具速查

### 4.1 债权融资

| 工具 | 额度 | 期限 | 成本参考 | 特点 |
|------|------|------|---------|------|
| **银行流动资金贷款** | 视抵押物 | 1-3年 | 3.5-5% | 需要抵押或担保 |
| **银行承兑汇票** | 保证金+敞口 | 6-12个月 | 手续费0.5‰ | 可背书转让支付供应商 |
| **应收账款保理** | 应收款的70-90% | 按账期 | 年化6-10% | 释放应收款占用 |
| **融资租赁** | 设备价值 | 2-5年 | 年化6-12% | 以租代买，设备做担保 |
| **过桥资金** | 视情况 | 1-3个月 | 日息1-3‰ | **昂贵**，只做短期周转 |
| **小微企业贷款** | 10-500万 | 1年 | 3-5%（含贴息） | 政府贴息、担保基金支持 |

### 4.2 股权融资

| 阶段 | 典型融资额 | 稀释比例 |
|------|-----------|---------|
| 天使轮 | 50-500万 | 10-20% |
| Pre-A / A轮 | 500-5000万 | 15-25% |
| B轮+ | 5000万以上 | 10-20% |

> 股权融资成本看似低，实际是出让未来收益权，长期看比债权贵。

## 五、紧急情况应对

### 5.1 现金流危机三级响应

| 级别 | 信号 | 措施 |
|------|------|------|
| **一级（预警）** | 经营现金流连续2月为负 | 暂停非必要采购、加强催款、压缩费用 |
| **二级（紧张）** | 可用资金 < 2个月固定支出 | 暂停招聘、协商推迟付款、申请贷款展期、寻求过桥资金 |
| **三级（危机）** | 可用资金 < 1个月工资+税款 | 裁员/降薪（最后手段）、变卖非核心资产、与债权人协商债务重组 |

### 5.2 永远不能动的钱

- ✅ 税款 — 有法定期限，滞纳金日万分之五
- ✅ 工资 — 《劳动法》保护，拖欠可被仲裁
- ✅ 社保 — 员工基本保障

> 先保工资+税款+社保，再考虑供应商、房租等弹性支出。
