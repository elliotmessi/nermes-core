# 资产管理技能

## 技能描述

资产管理技能用于企业固定资产和无形资产的全生命周期管理，包括资产采购入库、领用转移、折旧摊销、盘点清查、维修保养和处置报废等环节。系统为每项资产建立电子档案（资产卡片），记录其规格型号、使用部门、存放地点、价值变动等信息。支持条码/RFID标签管理、定期盘点计划制定与执行、资产使用效率分析等功能。

## 适用场景

- **资产入库管理**：新购资产的验收登记、卡片建立和编码管理
- **资产日常管理**：部门间资产调拨、借用归还、维修保养记录等
- **资产清查盘点**：按年度/季度制定盘点计划，使用移动端扫码进行现场盘点
- **资产处置决策**：为即将到寿命期的资产提供处置建议（出售、报废、捐赠等）

## 输入格式

```json
{
  "operation": "asset_acquisition",
  "asset_category": "办公设备",
  "asset_name": "激光打印机",
  "specification": "HP LaserJet Pro M404dn",
  "department": "行政部",
  "location": "3层办公区",
  "quantity": 2,
  "unit_price": 3500.00,
  "acquisition_date": "2026-05-20",
  "supplier": "HP授权经销商",
  "useful_life_years": 5,
  "depreciation_method": "直线法"
}
```

## 注意事项

- 资产卡片信息必须与实物一致，资产发生变动后需及时更新卡片
- 固定资产标准（价值和使用年限）应符合企业会计制度规定，一般单价2000元以上且使用年限超过一年
- 资产盘点发现的盘盈、盘亏需查明原因，经审批后做账务处理
- 融资租入固定资产应视同自有资产管理，经营租入资产需在备查簿中登记
- 资产减值测试建议每年度进行一次，出现减值迹象时需及时计提减值准备
