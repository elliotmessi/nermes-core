# 数据分析技能 — 店铺数据诊断与优化

## 功能描述
对店铺运营数据进行系统分析，包括流量结构、转化漏斗、商品表现、客户画像等，帮助运营者发现问题、定位瓶颈、制定优化策略。

## 核心能力

### 1. 流量分析
- **流量渠道拆解**：免费流量（搜索/首页/淘客/内容） vs 付费流量（直通车/引力魔方/万相台） vs 活动流量
- **渠道转化效率**：各渠道 UV、CVR、客单价、ROI 对比
- **流量健康度**：是否有单一渠道依赖风险、免费流量占比
- **趋势分析**：7日/30日环比、同比变化

### 2. 商品分析
- **贡献度排行**：按 GMV、利润、流量贡献排序
- **动销率分析**：在售商品中30天内有成交的比例
- **商品生命周期**：新品孵化期、成长期、成熟期、衰退期
- **价格带分析**：各价格区间的销量和转化情况

### 3. 转化漏斗
- **全链路漏斗**：曝光 → 点击 → 访问 → 收藏加购 → 下单 → 支付 → 确认收货
- **流失节点定位**：识别流失最严重的环节（如加购到下单的转化率低）
- **竞品转化对比**：同行业同品类的平均转化率参考

### 4. 客户分析
- **新老客占比**：首次购买 vs 复购比例
- **客户画像**：性别、年龄、地域、消费能力
- **复购率**：30天/90天复购率，会员回购率
- **客单价**：件单价、连带率、客单价趋势

### 5. 竞品监控
- **价格波动**：关注竞品调价策略
- **销量预估**：基于评价数/月销估算
- **流量结构**：判断竞品主要流量来源

## 使用场景
- 周/月经营报表分析
- 流量下降原因的排查
- 商品是否成为爆款的判断
- 广告投放效果复盘
- 竞品追踪分析

## 输入
店铺后台数据（生意参谋/商智/多多参谋等导出CSV）、竞品链接（可选）、分析周期

## 输出
数据分析报告、可视化图表建议、优化优先级排序、具体行动建议
