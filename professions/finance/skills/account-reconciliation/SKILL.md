# 银行对账

## 概述

本技能涵盖企业银行账户对账的全流程，包括银行余额调节表的编制方法、未达账项的分类处理、以及账实差异的系统性分析。定期对账是确保银行存款账务准确、防范资金风险的关键环节。

## 银行余额调节表编制

### 基本原理

银行余额调节表用于核对**企业银行存款日记账余额**与**银行对账单余额**之间的差异。核心公式：

```
企业日记账余额 + 银收企未收 - 银付企未付 = 调节后余额
银行对账单余额 + 企收银未收 - 企付银未付 = 调节后余额
```

两个调节后的余额应当相等。

### 编制步骤

1. **获取数据**：导出企业日记账余额及银行对账单余额
2. **逐笔核对**：按银行流水号、金额、日期逐笔勾对
3. **标记差异**：标识已入企业账未入银行账（未达）及已入银行账未入企业账的明细
4. **编制调节表**：按公式填入数据，计算调节后余额
5. **验证平衡**：确认两个调节后余额相等

## 未达账项处理

### 未达账项分类

| 类别 | 说明 | 常见场景 |
|------|------|---------|
| **企收银未收** | 企业已记收入，银行尚未入账 | 存现未到账、转账在途 |
| **企付银未付** | 企业已记支出，银行尚未扣款 | 已开支票对方未兑现 |
| **银收企未收** | 银行已入账，企业尚未记录 | 银行利息、自动托收 |
| **银付企未付** | 银行已扣款，企业尚未记账 | 手续费、代扣水电费 |

### 处理时限建议
- 企收银未收：正常 1-2 个工作日到账，超过 3 个工作日应查询
- 企付银未付：支票有效期 10 天（部分银行更长），超期需作废重开
- 银收企未收：应在取得回单当日入账
- 银付企未付：应在网银查询到扣款当日入账

## 差异分析

### 差异类型与排查思路

| 差异类型 | 可能原因 | 排查方法 |
|---------|---------|---------|
| **时间性差异** | 未达账项 | 编制调节表后差异消除 |
| **金额差异** | 记账错误、尾差 | 逐笔核对金额，检查小数位 |
| **笔数差异** | 漏记/重复记账 | 分别汇总双方笔数，定位遗漏 |
| **科目串户** | 款项记入其他账户 | 检查辅助核算余额 |
| **银行差错** | 银行误记 | 提供回单联系银行更正 |

### 异常差异分析表

```sql
-- 分析长期未达账项
SELECT bank_account, transaction_date, amount, description,
       DATEDIFF(CURDATE(), transaction_date) AS days_outstanding
FROM reconciliation_adjustments
WHERE cleared = FALSE
  AND DATEDIFF(CURDATE(), transaction_date) > 30
ORDER BY days_outstanding DESC;
```

### 长期未达账项处理
- **超过 1 个月**：标记为关注，联系银行确认
- **超过 3 个月**：升级为异常，发函询证
- **超过 6 个月**：评估是否需要调整账务（如计提损失）

## 自动化对账方案

### 规则引擎
```python
def auto_reconcile(bank_entries: list, book_entries: list) -> dict:
    """自动匹配银行流水与企业日记账"""
    matched = []
    unmatched_bank = []
    unmatched_book = []

    # 精确匹配：同金额、同日期、同流水号
    for be in bank_entries:
        for je in book_entries:
            if (abs(be.amount - je.amount) < 0.01
                    and be.date == je.date
                    and be.ref_no == je.ref_no):
                matched.append((be, je))
                continue

    # 模糊匹配：同金额、日期±2天
    # 标记剩余未匹配项
    return {
        'matched_count': len(matched),
        'bank_unmatched': unmatched_bank,
        'book_unmatched': unmatched_book
    }
```

## 最佳实践

1. **月度必对**：每月收到银行对账单后 5 个工作日内完成对账
2. **电子对账**：开通网银电子对账功能，支持自动导入流水
3. **余额调节表归档**：逐月保存，审计备查
4. **大额资金当日核**：超过 50 万元的收支当日逐笔确认
5. **差异闭环管理**：每项差异需注明原因、处理人、处理日期
6. **对账岗位分离**：制单与对账人员分离，确保内部控制有效性
