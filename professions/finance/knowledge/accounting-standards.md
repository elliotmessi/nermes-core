# 中国企业会计准则（CAS）核心要点

> 基于财政部2006年发布、历经多次修订（2017—2023年）的现行会计准则体系。适用于在大陆注册的企业财务报表编制。

---

## 一、会计准则体系概述

中国企业会计准则体系（CAS）由三部分构成：

| 层级 | 内容 | 说明 |
|-----|------|------|
| **基本准则** | 财务报告目标、会计假设、会计信息质量要求、要素定义 | 会计准则的"宪法" |
| **具体准则** | 42项具体准则（CAS 1—CAS 42） | 各项业务的具体处理规则 |
| **应用指南/解释** | 财政部发布的解释公告和实务指引 | 对具体准则的操作说明 |

**适用范围**：上市公司必须执行CAS；大中型企业鼓励执行；小企业可执行《小企业会计准则》。

---

## 二、会计基本假设与原则

### 2.1 四大基本假设

| 假设 | 含义 |
|-----|------|
| **会计主体** | 会计核算以企业自身为边界，与企业所有者个人财务严格区分 |
| **持续经营** | 假定企业不会在可预见的将来破产清算 |
| **会计分期** | 将持续经营的活动分为年度、季度、月度等期间 |
| **货币计量** | 以人民币为记账本位币（业务以外币为主的可选定外币，但报表须折算为人民币） |

### 2.2 会计信息质量要求

| 要求 | 含义 |
|-----|------|
| **可靠性** | 以实际发生的交易为依据，如实反映 |
| **相关性** | 信息对使用者的经济决策有用 |
| **可理解性** | 清晰明了，便于使用者理解 |
| **可比性** | 同一企业不同时期、不同企业同一时期的方法一致 |
| **实质重于形式** | 按经济实质而非法律形式确认（如融资租赁视同购买） |
| **重要性** | 重要事项充分披露，次要事项可简化处理 |
| **谨慎性** | 不高估资产/收益，不低估负债/费用 |
| **及时性** | 及时确认、及时披露 |

## 三、核心会计准则详解

### 3.1 CAS 14——收入（2017年修订）

**核心原则**：客户取得商品或服务的**控制权**时确认收入。

**五步法模型**（与IFRS 15趋同）：

| 步骤 | 操作 | 通俗理解 |
|-----|------|---------|
| **第一步**：识别客户合同 | 判断是否存在书面/口头/默示合同，双方已批准、有明确权利和义务、有支付条款、具有商业实质、对价很可能收回 | 先看看买卖双方的"协议"是否成立 |
| **第二步**：识别合同中的履约义务 | 拆解合同中承诺向客户交付的"每一项单独的商品或服务" | 比如卖设备+提供安装培训，算一项还是两项义务？ |
| **第三步**：确定交易价格 | 合同约定金额，考虑可变对价、重大融资成分、非现金对价、应付客户对价 | 最终能收多少钱？（折扣、返利、奖励都要考虑） |
| **第四步**：将交易价格分摊至各履约义务 | 按单独售价的相对比例分摊 | 设备值18万、培训值2万，总价19万→设备摊17.1万，培训摊1.9万 |
| **第五步**：在履行各履约义务时确认收入 | 在某一时点或某一时段内确认 | 发货签收时确认？还是分月确认？ |

**时点 vs 时段确认**：

| 确认方式 | 条件 | 示例 |
|---------|------|------|
| **某一时点**（默认） | 客户取得控制权（签收/验收） | 普通商品销售、设备销售 |
| **某一时段内** | 客户在企业履约的同时即取得并消耗；或客户能控制在建资产；或产品定制且有权收取成本+合理毛利 | 建筑施工、软件开发服务、长期服务合同 |

### 3.2 CAS 21——租赁（2018年修订）

**核心变化**：承租人不再区分经营租赁和融资租赁，统一采用**单一模型**（使用权资产 + 租赁负债）。

**承租人会计处理**（2021年1月1日起执行）：

**初始确认**：
```
借：使用权资产   （租赁付款额现值 + 初始直接费用 + 拆除复原成本）
    租赁负债——未确认融资费用（差额）
贷：租赁负债——租赁付款额  （各期租金总额）
    银行存款            （初始直接费用等）
```

**后续计量**：
- 使用权资产：按成本减累计折旧（直线法）及减值损失
- 租赁负债：按**实际利率法**摊销，确认财务费用
- 每期支付租金：
```
借：租赁负债——租赁付款额
贷：银行存款

借：财务费用
贷：租赁负债——未确认融资费用
```

**短期租赁（≤12个月）& 低价值资产租赁（≤5,000元）**：可简化处理，直接确认为当期费用（不确认使用权资产和租赁负债）。

**出租人**：仍区分经营租赁和融资租赁，处理与旧准则基本一致。

### 3.3 CAS 22——金融工具确认与计量（2017年修订）

**核心变化**：金融资产从原来的"四分类"改为"三分类"（与IFRS 9趋同）。

**金融资产三分类**：

| 分类 | 判断标准 | 计量方式 | 常见示例 |
|------|---------|---------|---------|
| **(1) 摊余成本计量** | 业务模式为"收取合同现金流量"且合同现金流仅为本金+利息 | 摊余成本法 | 普通贷款、应收账款、持有至到期的债券 |
| **(2) 以公允价值计量且变动计入其他综合收益（FVOCI）** | 业务模式为"收取+出售"，且现金流仅为本金+利息 | 公允价值变动→其他综合收益（OCI）；处置时转入损益 | 计划持有+择机出售的债券投资 |
| **(3) 以公允价值计量且变动计入当期损益（FVTPL）** | 不符合前两类的所有金融资产（含权益投资） | 公允价值变动→当期损益（利润表） | 股票投资、基金、交易性债券、衍生工具 |

> **权益工具的特殊选择**：非交易性权益投资（如上市公司股票、非上市公司股权）可在初始确认时**不可撤销地**指定为FVOCI，后续公允价值变动计入其他综合收益且**处置时不再转入损益**（仅股利收入进利润表）。

**金融资产减值——预期损失模型（ECL）**：

| 阶段 | 判断标准 | 减值计量 |
|-----|---------|---------|
| 第一阶段 | 信用风险未显著增加 | 未来12个月的预期信用损失 |
| 第二阶段 | 信用风险显著增加但未发生信用减值 | 整个存续期的预期信用损失 |
| 第三阶段 | 已发生信用减值 | 整个存续期的预期信用损失，利息按摊余成本计算 |

## 五、财务报表体系

企业应编制四张主表和一个附注：

| 财务报表 | 结构要点 |
|---------|---------|
| **资产负债表** | 资产 = 负债 + 所有者权益（某一时点的财务"照片"） |
| **利润表** | 收入 - 费用 = 净利润（一个期间的经营"录像"） |
| **现金流量表** | 经营/投资/筹资三类现金流（反映钱的"来龙去脉"） |
| **所有者权益变动表** | 实收资本、资本公积、盈余公积、未分配利润的变化 |
| **附注** | 会计政策、估计、或有事项、关联交易等的详细说明 |

## 六、CAS与IFRS的主要差异

| 差异项 | CAS | IFRS |
|-------|-----|------|
| 资产减值转回 | 长期资产（固定资产、无形资产）减值**不得转回** | 允许转回（商誉除外） |
| 存货计量 | 禁止使用后进先出法（LIFO） | 允许LIFO |
| 政府补助 | 总额法/净额法均可 | 原则上总额法 |
| 关联方披露 | 国有企业之间不视为关联方（特殊豁免） | 无此豁免 |
| 财务报表格式 | 财政部统一格式，变更需批准 | 格式灵活，仅要求列报特定项目 |

## 七、常见实务问题速答

**Q1：小企业可以执行小企业会计准则吗？**
A：可以。符合《中小企业划型标准规定》的小企业可执行《小企业会计准则》，会计核算更简化（如不要求资产减值、不要求公允价值计量）。

**Q2：会计政策变更和会计估计变更有什么区别？**
A：会计政策变更（如折旧方法改变）需追溯调整；会计估计变更（如折旧年限改变）采用未来适用法，不需调整以前年度。

**Q3：什么情况要做合并报表？**
A：母公司拥有一个或多个子公司的控制权（通常持股>50%或拥有实际控制权）时，需编制合并财务报表。

**Q4：研发支出什么时候资本化？**
A：**研究阶段**全部费用化；**开发阶段**满足五个条件（技术上可行、有意完成、能使用/出售、有资源支撑、支出能可靠计量）的才能资本化确认为无形资产。
