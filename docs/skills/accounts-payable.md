# 应付账款管理技能

## 技能描述

应付账款管理技能用于企业对供应商应付款项的全流程管理，包括采购订单匹配、发票核验、付款计划制定、付款执行和账龄分析等功能。系统通过与采购系统和发票系统的数据对接，自动完成三单匹配（采购订单、入库单、发票），确保付款金额准确无误。提供供应商对账、付款预测、资金计划编制和信用评估等高级功能。

## 适用场景

- **采购付款管理**：根据采购合同和入库情况，安排供应商付款并跟踪执行状态
- **账龄分析与付款排期**：按供应商账期和优先级制定付款计划，优化现金流出
- **供应商对账**：定期与供应商核对应付余额，处理差异并生成对账确认函
- **现金折扣管理**：自动识别供应商提供的现金折扣条件，计算最优付款方案

## 输入格式

```json
{
  "operation": "payment_scheduling",
  "supplier_code": "SUP001",
  "supplier_name": "XX电子元器件有限公司",
  "payment_terms": "30天",
  "cash_discount": "2/10, Net 30",
  "invoices": [
    {"invoice_no": "INV-2026-001", "amount": 120000.00, "due_date": "2026-06-15"}
  ],
  "available_funds": 500000.00,
  "payment_priority": "high"
}
```

## 注意事项

- 付款前务必完成三单匹配，确保采购、入库和发票信息一致
- 关注供应商付款账期，避免过早付款占用资金或逾期付款影响供应链关系
- 现金折扣需要计算实际年化收益率，合理决策是否提前付款
- 应付账款周转天数应结合行业特点和供应商关系策略综合管理
- 大额付款需执行双人审批制，防范资金挪用风险
