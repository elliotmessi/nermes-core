# 财务比率分析技能

## 技能描述

财务比率分析技能用于对企业财务状况和经营成果进行量化分析，通过计算一系列财务比率指标，评估企业的偿债能力、营运能力、盈利能力和发展能力。系统从财务报表自动提取数据，计算四大类共20余个核心财务比率，并支持与行业标准值、历史同期值和预算目标值进行多维度对比分析，输出直观的雷达图、趋势图和文字分析结论。

## 适用场景

- **企业健康诊断**：定期对企业的财务健康度进行全面体检，发现潜在问题
- **投资决策支持**：投资者通过财务比率分析评估目标企业的投资价值
- **银行信贷审批**：银行在审批贷款时分析企业的偿债能力和现金流状况
- **竞争对手对标**：将自身财务比率与行业标杆企业对比，找出差距和改进方向

## 输入格式

```json
{
  "company_code": "C001",
  "company_name": "XX科技有限公司",
  "industry": "软件和信息技术服务业",
  "period": "2026-Q1",
  "financial_statements": {
    "balance_sheet_path": "/data/fs/2026Q1_balance_sheet.json",
    "income_statement_path": "/data/fs/2026Q1_income_statement.json",
    "cash_flow_path": "/data/fs/2026Q1_cash_flow.json"
  },
  "benchmark_data": {
    "industry_avg_path": "/data/benchmark/software_industry_2025.json"
  }
}
```

## 注意事项

- 财务比率分析需要与企业的具体行业和商业模式结合，不同行业的合理区间差异很大
- 单一比率不能全面反映企业状况，应结合多个比率和绝对指标综合分析
- 季节性业务的企业进行环比分析时需消除季节性波动的影响
- 会计政策变更会导致同企业前后期比率口径不一致，分析时需调整
- 资产负债表是时点数据，利润表是期间数据，计算涉及两者结合的比率时需注意口径统一
