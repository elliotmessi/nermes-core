# 税务风险扫描技能

本技能提供税务风险识别与评估全流程指导，帮助企业在发票管理、税负率监控、关联交易、纳税调整等方面主动发现和防范税务风险。

---

## 1. 税务风险总览

### 1.1 风险分类体系

```
税务风险
├── 发票风险（虚开/异常/失控发票）
├── 申报风险（税负率异常/零申报/逾期申报）
├── 关联交易风险（转移定价/特别纳税调整）
├── 纳税调整风险（扣除不合规/优惠条件不满足）
├── 税务稽查风险（被列入重点稽查对象）
├── 国际税务风险（跨境交易/常设机构）
└── 其他风险（社保/个税/印花税等）
```

### 1.2 风险评级标准

| 风险等级 | 判定标准 | 建议应对措施 |
|----------|----------|-------------|
| 低 | 偶尔超标准扣除，金额较小 | 自行校正，次月调整 |
| 中 | 税负率偏离行业均值 20%-50%，有政策理解偏差 | 内部自查，补充备查资料 |
| 高 | 税负率异常 50%+，涉及虚开发票嫌疑，关联交易无合规文档 | 立即聘请税务顾问，准备应对稽查 |

## 2. 发票风险扫描

### 2.1 常见发票风险点

| 风险类型 | 具体表现 | 风险等级 |
|----------|----------|----------|
| 虚开发票 | 无真实交易开具/取得发票；票货款不一致 | 🔴 高 |
| 异常凭证 | 开票方走逃失联、发票被列为异常凭证 | 🔴 高 |
| 失控发票 | 开票方未报税或系统认定为失控 | 🔴 高 |
| 红字发票不规范 | 取得红字发票未及时进项转出 | 🟡 中 |
| 发票备注栏缺失 | 特定行业发票备注栏未按要求填写 | 🟢 低 |
| 发票内容错误 | 品名、数量、金额与合同不一致 | 🟡 中 |
| 发票税率错误 | 适用税率与实际业务不符 | 🟡 中 |
| 发票认证不及时 | 大量未认证/逾期认证进项发票 | 🟢 低 |

### 2.2 发票风险核查清单

**每月自查：**
- [ ] 核对进项发票与付款凭证、合同是否一致（三流一致）
- [ ] 检查是否存在同一客户短期内大量红冲又重开
- [ ] 检查是否存在月底集中开票、发票单价异常高/低
- [ ] 检查是否有陌生企业开具的大额专票
- [ ] 检查供应商是否在正常经营状态（可通过企查查/天眼查核实）

**季度自查：**
- [ ] 核查进项发票是否全部在税务局系统有对应认证记录
- [ ] 核查红字发票是否已全部做进项税额转出
- [ ] 检查是否存在"富余票"（收入不开票但进项已抵扣的情况）

### 2.3 异常凭证应对

**收到异常凭证通知时：**
1. 立即暂停申报抵扣或出口退税。
2. 10 个工作日内向主管税务机关提交证明材料：
   - 合同/协议复印件
   - 付款凭证（银行回单）
   - 物流单据或出入库单
   - 能够证明业务真实性的其他资料
3. 如确属真实交易，税务机关应在 20 个工作日内核实并恢复凭证正常状态。
4. 如无法证明真实性，需做进项税额转出并补缴税款。

### 2.4 三流一致性检查

```
资金流：付款方 = 发票购买方 = 合同买方
发票流：开票方 = 合同卖方 = 收款方
货物流：发货方 = 合同卖方 = 开票方

每条记录必须能对应到同一笔交易。
```

> **风险提示**：三方抵账协议、代付款特例要保留完整的书面证据链。

## 3. 税负率异常风险

### 3.1 税负率计算公式

```
增值税税负率 = 当期应纳增值税 ÷ 当期应税销售收入 × 100%
企业所得税税负率 = 当期应纳所得税额 ÷ 当期利润总额 × 100%
综合税负率 = 实际缴纳的全部税款 ÷ 营业收入 × 100%
```

### 3.2 行业参考税负率

> **注意**：以下数据为行业平均参考值，各地税务机关统计口径可能不同，具体以当地税务机关发布的行业税负标准为准。

| 行业 | 增值税税负率参考 | 所得税税负率参考 |
|------|-----------------|-----------------|
| 制造业 | 2.5%-4.5% | 1.5%-3.0% |
| 批发零售 | 0.8%-2.5% | 0.5%-1.5% |
| 建筑业 | 2.0%-3.5% | 1.0%-2.5% |
| 房地产业 | 3.5%-5.5% | 2.0%-4.0% |
| 信息技术 | 1.5%-3.0% | 2.0%-4.0% |
| 租赁和商务服务 | 1.5%-3.5% | 1.5%-3.0% |
| 交通运输 | 1.5%-3.0% | 1.0%-2.0% |

### 3.3 税负率异常判定

```
税负率偏差 = (企业税负率 - 行业平均税负率) ÷ 行业平均税负率 × 100%

偏差 ≤ -50%  → 🔴 高风险（可能少交税或进项异常）
-50% < 偏差 ≤ -20% → 🟡 中风险（需要合理解释）
-20% < 偏差 < 20% → 🟢 正常范围
偏差 ≥ 50% → 🟡 中风险（可能多交税或错用税率）
```

### 3.4 税负率异常常见原因

**偏低原因及应对：**
| 原因 | 应对措施 |
|------|----------|
| 大量固定资产进项抵扣 | 撰写说明，提供采购合同及认证清单 |
| 库存积压导致进项大 | 核查库存账实是否一致 |
| 错误抵扣进项（集体福利等不可抵扣项目）| 自查并进项转出 |
| 关联交易导致定价偏低 | 准备转移定价同期资料 |
| 适用了错误的低税率 | 重新确认业务适用的正确税率 |

**偏高原因及应对：**
| 原因 | 应对措施 |
|------|----------|
| 进项发票未及时认证 | 建立发票认证台账，确保及时勾选 |
| 可抵扣进项较少 | 检查能否取得合规专票 |
| 未充分享受税收优惠 | 匹配可享受的政策，补申请 |
| 实际毛利率较高 | 正常经营所致，留存备查 |

## 4. 关联交易风险

### 4.1 关联关系认定

符合以下任一条件的构成关联关系：
- 股权控制：一方直接或间接持有另一方 25%+ 股份
- 资金借贷：一方借贷资金占另一方注册资本 50%+ 或借贷总额 10%+
- 经营控制：一方对另一方的经营决策具有实质控制权
- 人员关联：一方的董事或高管 50%+ 由另一方委派
- 家族关联：双方主要股东为亲属关系

### 4.2 转移定价风险点

| 风险行为 | 表现 | 风险等级 |
|----------|------|----------|
| 关联交易价格偏离市场价 | 向关联方低价销售/高价采购 | 🔴 高 |
| 跨境关联交易无文档 | 未准备国别报告/主体文档/本地文档 | 🔴 高 |
| 无形资产转让定价不公允 | 技术/商标使用费过高或不合理 | 🟡 中 |
| 关联劳务费用不合理 | 未接受实际劳务却支付管理服务费 | 🟡 中 |
| 资本弱化 | 关联方债权与权益比例超标（金融企业 5:1，其他 2:1）| 🟡 中 |

### 4.3 合规要求

**关联交易申报：**
- 年度汇算清缴时填报《企业年度关联业务往来报告表》（国别报告+主体文档+本地文档）。
- 关联交易金额达到以下门槛需准备同期资料：
  - 年度关联交易总额 > 10 亿元 → 需准备主体文档 + 本地文档
  - 年度关联购销金额 > 2 亿元 或 其他关联交易 > 4000 万元 → 需准备本地文档
  - 跨境关联交易需关注国别报告义务

**转让定价方法：**
| 方法 | 适用场景 |
|------|----------|
| 可比非受控价格法 | 有形资产购销 |
| 再销售价格法 | 分销商关联交易 |
| 成本加成法 | 关联劳务/制造 |
| 交易净利润法 | 综合性关联交易 |
| 利润分割法 | 高度整合的关联交易 |

### 4.4 关联交易风险评分

```
风险评分 = 0
IF 有关联购销 → +20
IF 有关联借贷且债资比超标 → +25
IF 跨境关联交易 → +30
IF 无同期资料 → +35
IF 关联方在低税率地区（如开曼/维京群岛）→ +30
IF 关联交易定价无定价政策文档 → +20

评分 ≥ 80 → 🔴 高风险，建议聘请专业税务顾问
50 ≤ 评分 < 80 → 🟡 中风险，尽快补充同期资料
评分 < 50 → 🟢 低风险，维持日常监控
```

## 5. 纳税调整风险

### 5.1 常见纳税调整风险点

| 调整项目 | 企业常见错误 | 风险说明 |
|----------|-------------|----------|
| 业务招待费 | 全额扣除 | 应调增（按 60% 与 5‰ 孰低）|
| 广告宣传费 | 超 15% 未调增 | 超出部分应结转 |
| 公益性捐赠 | 超利润 12% 未调增 | 超出部分可结 3 年 |
| 工资薪金 | 未实际支付提前扣除 | 次年 5 月 31 日前未支付的调增 |
| 职工福利费 | 超 14% 未调增 | 超支调增 |
| 工会经费 | 未取得工会收据 | 未取得《工会经费收入专用收据》的调增 |
| 职工教育经费 | 超 8% 未调增 | 超支可结转 |
| 资产减值准备 | 全额扣除 | 税法不允许扣除，需调增 |
| 公允价值变动损益 | 按会计利润计税 | 未实现收益调减/损失调增 |
| 长期待摊费用 | 摊销年限不符 | 税法有最低摊销年限规定 |

### 5.2 优惠条件符合性检查

**享受优惠时的附加风险：**
- 高新技术企业：核心知识产权是否有效？科技人员占比是否持续满足？
- 研发费用加计扣除：研发项目是否属于负面清单行业？费用归集是否准确？
- 小微企业：资产总额/从业人数是否超标？与关联企业的合并计算要求？
- 软件企业：软件收入占比、研发费用比例是否持续达标？

### 5.3 核查触发点

> 以下情况会触发税务机关重点关注：
1. 连续 6 个月零申报/负申报
2. 增值税税负率明显偏离同行业
3. 大量红字发票（红冲率 > 20%）
4. 大额异地采购且无合理解释
5. 发票开具金额突增（超过上期 300%）
6. 存在大量作废发票
7. 年度汇算清缴大面积退税
8. 供应商/客户集中在特定高风险区域

## 6. 税务稽查风险

### 6.1 稽查对象筛选机制

税务机关确定稽查对象的主要依据（以金税四期系统数据为主）：
1. **税收风险模型评分**：综合评估企业风险分值，分值越高越可能被稽查。
2. **行业专项稽查**：每年会选取 1-3 个行业开展专项稽查。
3. **举报线索**：来自举报、协查等线索。
4. **随机抽查**："双随机、一公开"机制下按比例随机抽取。
5. **协查任务**：异地税务机关发起的协查请求。

### 6.2 稽查应对流程

```
[收到稽查通知]
     ↓
[确认稽查范围和期间]
     ↓
[整理备查资料]
     ↓
[检查风险点→自我评估]
     ↓
[配合稽查→提供资料]
     ↓
[确认稽查结果]
     ↓
[如有异议→申请行政复议/提起行政诉讼]
```

### 6.3 稽查备查资料清单

| 类别 | 资料清单 |
|------|----------|
| 证照类 | 营业执照、税务登记证、行业许可证、各类认定证书 |
| 财务类 | 会计凭证、总账明细账、财务报表、审计报告 |
| 申报类 | 各期纳税申报表及附表、完税证明 |
| 发票类 | 发票领用簿、发票存根联、抵扣联、发票认证清单 |
| 合同类 | 采购合同、销售合同、关联交易合同 |
| 人事类 | 员工花名册、社保缴费记录、工资表、考勤记录 |
| 研发类 | 研发项目立项书、研发费用辅助账、知识产权证明 |
| 资产类 | 固定资产台账、折旧计提表、资产减值测试表 |

### 6.4 自查自纠

**自查要点：**
1. 核对各期申报数据与账面数据是否一致。
2. 检查是否存在未申报的应税收入。
3. 核查费用扣除凭证是否合规、齐全。
4. 核对发票信息与纳税申报表的一致性。
5. 检查享受优惠的资格条件是否仍然满足。

**自纠原则：**
- 主动纠正优于被动查处：主动补税可减轻或免除罚款。
- 补缴税款时同时补缴滞纳金（每日万分之五）。
- 主动披露（主动报告税务机关发现的重大风险点）可申请从轻处理。

## 7. 税务风险监控体系建设

### 7.1 日常监控指标

| 指标 | 监控频率 | 预警阈值 |
|------|----------|----------|
| 增值税税负率 | 月度 | 偏离行业均值 ±20% |
| 企业所得税贡献率 | 季度/年度 | 低于行业下限 |
| 发票红冲率 | 月度 | > 20% |
| 异常发票数量 | 月度 | 任何异常 → 立即关注 |
| 零申报/负申报次数 | 月度 | 连续 3 个月 |
| 关联交易金额变动 | 季度 | 同比变动 > 30% |
| 研发费用占比变动 | 年度 | 大幅下降 |
| 汇算清缴退税率 | 年度 | 大幅退税 |

### 7.2 风险预警通知机制

```
IF 某指标触发预警阈值 THEN
    自动列入"税务风险监控清单"
    通知财务负责人 → 启动自查程序
    IF 风险确认 THEN
        制定整改计划
        必要时聘请税务顾问
    END IF
END IF
```

### 7.3 年度风险评估报告模板

**结构建议：**
1. **总体评估**：年度税务风险等级（低/中/高）。
2. **各维度风险分析**：
   - 发票管理风险情况
   - 申报合规风险情况
   - 纳税调整风险情况
   - 关联交易风险情况
   - 政策享受合规情况
3. **发现的问题及整改建议**。
4. **下年度风险防控计划**。

## 8. 常见风险案例

### 案例 1：发票三流不一致
**情况**：某采购部门通过中间人采购材料，发票由 B 公司开具，但款项付给了中间人个人账户。
**风险**：虚开发票嫌疑，发票被列为异常凭证。
**处理**：需提供由中间人出具的代收款证明+委托采购协议+B 公司确认函。如无法补充证明，需做进项税额转出。

### 案例 2：小型微利企业认定错误
**情况**：某集团下有多个子公司，从业人数/资产总额分散但未合并计算关联方指标。
**风险**：各子公司分别按小微申报，但合并后超过标准，面临补税+罚款。
**处理**：重新核实各子公司是否符合小微条件，如不符合应重新申报。

### 案例 3：跨境关联交易无同期资料
**情况**：某外商投资企业与其境外母公司发生管理服务费往来，年度金额 5000 万元，无转让定价文档。
**风险**：被特别纳税调整，补税+加收利息。
**处理**：立即准备本地文档，证明管理服务费定价符合独立交易原则。

## 9. 政策咨询渠道

- **国家税务总局 12366 纳税服务热线**：政策咨询、投诉举报。
- **电子税务局在线咨询**：实时在线问答。
- **主管税务机关专管员**：企业专属税务管理人员。
- **专业税务顾问/税务师事务所**：复杂事项的专业支持。
- **金税四期系统风险提示**：系统主动推送的涉税风险提示信息。
