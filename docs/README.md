## 什么是 Nermes？

**Nermes** 是一个现代化的 AI 个人助理框架，旨在帮助用户通过自然语言与各类工具和服务进行交互。无论你是想快速记账、查询天气、管理日程，还是构建一个自动化的个人工作流，Nermes 都能以极低的配置成本满足你的需求。

Nermes 采用**插件化架构**，核心引擎轻量高效，功能通过插件生态按需扩展。它支持多种部署方式（本地、云端、边缘设备），并提供丰富的平台集成能力（微信、Telegram、Discord 等），真正做到"一次配置，随处可用"。

## 核心功能

### 🧠 智能对话引擎
- 支持多种大语言模型（GPT-4o、Claude、DeepSeek、本地模型等）
- 上下文记忆管理，支持短时和长期记忆
- 多轮对话中的意图识别与任务路由

### 🔧 丰富的工具生态
- **财务管理**：记账、预算分析、账单提醒
- **信息查询**：天气、新闻、汇率、百科
- **日程管理**：日历事件创建、提醒设置、待办列表
- **文件操作**：读写文件、数据整理、格式转换
- **开发工具**：代码执行、Git 操作、API 调试
- **网络服务**：网页抓取、RSS 订阅、邮件收发

### 🔌 插件系统
- 即插即用：插件安装后自动发现，无需重启
- 社区市场：共享和发现由社区开发的插件
- 自定义开发：30 分钟即可编写一个自己的插件

### 🌐 多平台接入
- **即时通讯**：微信、Telegram、Discord、Slack
- **企业协作**：钉钉、飞书、企业微信
- **语音助手**：Home Assistant、智能音箱
- **Web API**：RESTful 接口，方便集成到自有系统

### 🔒 安全与隐私
- 本地优先：核心数据存储在你的设备上
- API Key 加密存储，不会明文传输
- 细粒度权限控制：每个工具可单独授权
- 支持完全离线运行（配合本地模型）

## 快速安装

### 系统要求

| 环境 | 最低要求 |
|------|----------|
| OS   | Linux / macOS / Windows (WSL2) |
| Python | 3.10 以上 |
| 内存 | 2GB（推荐 4GB+） |
| 磁盘 | 500MB（不含模型文件） |

### 安装方式

#### 方式一：pip 安装（推荐）

```bash
pip install nermes-core
```

#### 方式二：使用 Docker

```bash
docker pull nermes/nermes-core:latest
docker run -it --rm nermes/nermes-core
```

#### 方式三：从源码安装

```bash
git clone https://github.com/nermes/nermes-core.git
cd nermes-core
pip install -e .
```

### 启动验证

安装完成后，运行以下命令验证：

```bash
nermes --version
```

如果看到版本号输出，说明安装成功。接下来，请参考 [快速上手](quickstart.md) 完成配置。
